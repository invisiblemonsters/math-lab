**Natural Language:**
THEOREM: The sum of the elements of a list of natural numbers is equal to the sum of the elements of the list obtained by reversing the original list.

PROOF SKETCH: 
1. Use induction on the length of the list.
2. Base case: For a list with one element, reversing it doesn't change the sum.
3. Inductive step: Assume the property holds for lists of length n. For a list of length n+1, split it into the first element and the rest. Reversing the list is equivalent to appending the first element to the reversed rest. Show that the sum remains unchanged using the inductive hypothesis and the associativity of addition.

LEAN HINT:
- Imports: import Data.List
- Tactics: induction, simp, rw [list.reverse_cons, list.sum_cons], exact

DIFFICULTY: medium

**Lean 4 Proof:**
```lean
import Mathlib
import Data.List

theorem sum_reverse_eq_sum (l : List ℕ) : l.sum = l.reverse.sum :=
  induction l with
  | nil => rfl
  | cons a l ih =>
    simp [List.reverse_cons, List.sum_cons]
    rw [List.sum_cons, add_comm]
    exact ih
```
