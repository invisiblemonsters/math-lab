**Natural Language:**
THEOREM: For any natural numbers a, b, and c, if a + b = c, then b + a = c.

PROOF SKETCH: 
1. Start with the given equation a + b = c.
2. Use the commutative property of addition, which states that a + b = b + a.
3. Substitute b + a for a + b in the original equation, yielding b + a = c.
4. Conclude that the order of addition does not affect the result, thus proving the statement.

LEAN HINT:
- Imports: import Algebra
- Tactics: apply add_comm, exact, assumption

DIFFICULTY: easy

**Lean 4 Proof:**
```lean
import Mathlib

theorem add_comm_conclusion (a b c : Nat) (h : a + b = c) : b + a = c := 
  Eq.trans (Eq.symm (Nat.add_comm a b)) h
```
