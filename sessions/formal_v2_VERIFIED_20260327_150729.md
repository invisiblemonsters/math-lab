**Natural Language:**
THEOREM: The composition of a polynomial-time computable function with a polynomial-time decidable language is itself a polynomial-time decidable language. Formally, if f : Σ* → Σ* is a polynomial-time computable function and L ⊆ Σ* is a language in P, then the language { w | f(w) ∈ L } is also in P.

PROOF SKETCH:
1. Assume f is a polynomial-time computable function and L is a language in P.
2. By definition of P, there exists a polynomial p such that L can be decided in time p(|x|) for any input x.
3. Since f is polynomial-time computable, there exists a polynomial q such that f(x) can be computed in time q(|x|) for any input x.
4. To decide whether w is in { w | f(w) ∈ L }, compute f(w) in time q(|w|) and then decide if f(w) is in L in time p(|f(w)|).
5. The total time is bounded by q(|w|) + p(|f(w)|).
6. Since |f(w)| ≤ q(|w|) (as f runs in time q(|w|)), we have p(|f(w)|) ≤ p(q(|w|)).
7. Thus, the total time is bounded by q(|w|) + p(q(|w|)), which is a polynomial in |w|.
8. Therefore, { w | f(w) ∈ L } is in P.

theorem composition_in_P (f : Σ* → Σ*) (L : set Σ*) 
  (hf : poly_time_fun f) (hL : is_in_P L) : 
  is_in_P { w | f w ∈ L } :=
