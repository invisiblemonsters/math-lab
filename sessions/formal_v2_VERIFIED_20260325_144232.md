THEOREM: The image of a finite set of natural numbers under a function is also finite.

PROOF SKETCH:
1. **Definition Recall**: A set is finite if there exists a bijection with a natural number less than some natural number n.
2. **Function Application**: Given a function f: ℕ → ℕ and a finite set S, we need to show f(S) = { f(x) | x ∈ S } is finite.
3. **Finite Cardinality**: Since S is finite, it has a cardinality of n. The image f(S) can have at most n distinct elements (pigeonhole principle).
4. **Construct Surjection**: Define a surjective function from S to f(S), which exists by restricting f's domain to S.
5. **Finite Image**: Use the fact that the image of a finite set under any function is finite, as the surjection implies |f(S)| ≤ |S| < ∞.

import Mathlib
import Algebra.Polynomial

def is_polynomial (f : ℕ → ℕ) : Prop :=
  ∃ (p : Polynomial ℕ), ∀ (n : ℕ), f n = p.eval n

  cases hp with p_poly hp_eq,
  use (p_poly.comp (Polynomial.monomial 1 c + Polynomial.monomial 0 1)),
