THEOREM: For all natural numbers a, b, and c, if a ≤ b, then the minimum of (a + c) and (b + c) is equal to a + c.

PROOF SKETCH: 
1. **Assume** a ≤ b.
2. **Add** c to both sides, yielding a + c ≤ b + c by the monotonicity of addition.
3. **Apply** the definition of minimum: since a + c ≤ b + c, the minimum of (a + c, b + c) is a + c.

**Natural Language:**
THEOREM: For any natural numbers a, b, and c, if a ≤ b, then the minimum of (a + c) and (b + c) is equal to a + c.

PROOF SKETCH: 
1. Assume a, b, c ∈ ℕ and a ≤ b.
2. By the definition of ≤, there exists some d ∈ ℕ such that a + d = b.
3. Consider the expressions (a + c) and (b + c). Substitute b with (a + d) in the second expression to get (a + d + c).
4. Since addition is associative and commutative, (a + c) ≤ (a + d + c) because d is non-negative.
5. Therefore, min(a + c, b + c) = a + c.

LEAN HINT:
- Imports: import data.nat.basic
- Tactics: intro, apply, simp, omega

DIFFICULTY: easy

**Lean 4 Proof:**
```lean
import Mathlib

theorem min_add (a b c : Nat) (h : a ≤ b) : min (a + c) (b + c) = a + c := 
  by apply min_eq_left
     apply Nat.add_le_add_left
     exact h
```
