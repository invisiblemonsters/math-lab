THEOREM: For any natural numbers a, b, and c, if a ≤ b, then the minimum of (a + c) and (b + c) is equal to a + c.

PROOF SKETCH: 
1. Assume a, b, c ∈ ℕ and a ≤ b.
2. By the definition of ≤, there exists some d ∈ ℕ such that a + d = b.
3. Consider the expressions (a + c) and (b + c). Substitute b with (a + d) in the second expression to get (a + d + c).
4. Since addition is associative and commutative, (a + c) ≤ (a + d + c) because d is non-negative.
5. Therefore, min(a + c, b + c) = a + c.

LEAN HINT:
- Imports: import data.nat.basic
- Tactics: intro, apply, simp, omega

  { simp [List.range, List.sum] },
  { simp [List.range, List.sum, List.append, List.length, 
      List.map, List.sum_append, ih, Nat.add_comm, Nat.succ_eq_add_one],

theorem sum_strictly_increasing (m n : ℕ) (h : m < n) : 
  (List.range (m + 1)).sum < (List.range (n + 1)).sum := 
begin
  apply lt_of_le_of_lt,
  { have h1 : m * (m + 1) / 2 ≤ n * (n + 1) / 2,
    { apply Nat.div_le_div,
      { apply Nat.mul_le_mul,
        { exact Nat.le_of_lt h },
        { apply Nat.succ_le_succ,
          exact Nat.le_of_lt h } },
      { exact Nat.zero_lt_two } },
    rw [sum_first_m_natural_numbers m, sum_first_m_natural_numbers n],
    exact h1 },
  { have h2 : n * (n + 1) / 2 < n * (n + 1) / 2 + 1,
    { apply Nat.lt_add_one },
    apply lt_trans _ h2,
    apply Nat.div_lt_div,
    { apply Nat.mul_lt_mul_of_pos_left,
      { apply Nat.lt_add_one },
      { exact Nat.zero_lt (n + 1) } },
    { exact Nat.zero_lt_two } }
