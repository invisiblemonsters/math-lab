THEOREM: For any natural number n, the sum of the first n odd natural numbers is equal to the square of n. Formally, 1 + 3 + 5 + ... + (2n-1) = n².

PROOF SKETCH: 
We can prove this by mathematical induction. 
1. **Base case**: For n = 1, the left-hand side (LHS) is 1, and the right-hand side (RHS) is 1² = 1. Thus, the base case holds.
2. **Inductive step**: Assume the statement is true for some natural number k, i.e., 1 + 3 + 5 + ... + (2k-1) = k². We must show it holds for k+1.
   - The sum for k+1 is (1 + 3 + 5 + ... + (2k-1)) + (2(k+1)-1) = k² + (2k+1).
   - Simplifying, k² + 2k + 1 = (k+1)², which completes the inductive step.

LEAN HINT:

```lean
  induction n,
  { simp [finset.sum_empty] }, -- Base case
  { simp [finset.sum_range_succ, *] } -- Inductive step, using algebraic manipulation
end
```

PROOF SKETCH:
1. **Definition Recall**: Recall that a function f is polynomial-time computable if there exists a polynomial p such that the computation time of f(x) is bounded by p(|x|), where |x| denotes the length of input x.
2. **Composition Time Analysis**: Consider the composition h(x) = f(g(x)). The time taken to compute h(x) is the sum of the time to compute g(x) and the time to compute f(g(x)).
3. **Polynomial Bound**: Since f and g are polynomial-time computable, let p_f and p_g be their respective bounding polynomials. Then, the time to compute g(x) is ≤ p_g(|x|), and the time to compute f(g(x)) is ≤ p_f(|g(x)|).
4. **Output Length Bound**: Note that |g(x)|, the length of the output of g(x), is polynomially bounded in |x|. Specifically, there exists a polynomial q such that |g(x)| ≤ q(|x|) for all x.
5. **Combined Polynomial**: Substitute the bound on |g(x)| into the polynomial for f: p_f(q(|x|)) is still a polynomial in |x|. Thus, the total time for h(x) is ≤ p_g(|x|) + p_f(q(|x|)), which is a polynomial in |x|.
6. **Conclusion**: Therefore, h is polynomial-time computable.
