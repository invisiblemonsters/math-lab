**Natural Language:**
THEOREM: The sum of the first n natural numbers, denoted as 1 + 2 + ... + n, is equal to n(n + 1)/2.

PROOF SKETCH: We will use mathematical induction to prove this statement. 
1. **Base Case**: For n = 1, the left-hand side (LHS) is 1, and the right-hand side (RHS) is 1(1 + 1)/2 = 1. Thus, the base case holds.
2. **Inductive Step**: Assume the statement is true for some arbitrary natural number k, i.e., 1 + 2 + ... + k = k(k + 1)/2. We need to show it holds for k + 1. 
   - Starting with the sum up to k + 1: 1 + 2 + ... + k + (k + 1).
   - By the inductive hypothesis, this equals k(k + 1)/2 + (k + 1).
   - Factoring out (k + 1) gives (k + 1)(k/2 + 1) = (k + 1)(k + 2)/2, which matches the formula for n = k + 1.
