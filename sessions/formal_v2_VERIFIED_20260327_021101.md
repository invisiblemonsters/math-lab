THEOREM: The set of decidable languages is closed under the Kleene star operation. Formally, if a language L over an alphabet Σ is decidable, then the language L* (the set of all finite strings over Σ that can be formed by concatenating zero or more strings from L) is also decidable.

PROOF SKETCH:
1. **Definition Recall**: A language L is decidable if there exists a Turing machine (or an algorithm) that halts and accepts or rejects any input string.
2. **Kleene Star Construction**: To show L* is decidable, we construct a Turing machine that, given an input string w, checks all possible decompositions of w into a finite sequence of strings from L.
3. **Decidability Argument**: Since L is decidable, for each decomposition, we can verify if each part is in L. The finiteness of the input string ensures that there are only finitely many decompositions to check (specifically, the length of the input string plus one).
4. **Algorithm**: The Turing machine iterates over all possible decompositions, checking each part against L. If any decomposition is valid (all parts are in L), the machine accepts; otherwise, it rejects.
5. **Termination**: The algorithm terminates because the number of decompositions is finite and each check against L halts.

  { w | ∃ x, (x ++ w) ∈ L₁ }

-- Define a decidable language
def decidable_language (L : set (list Σ)) : Prop :=
  ∃ (M : turing_machine Σ bool), 
    (∀ (x : list Σ), M.eval x).dom ∧ 
    ∀ (x : list Σ), x ∈ L ↔ M.eval x = true

  (h : decidable_language L₁) : 
  decidable_language (left_quotient L₁) := 
begin

  -- Construct a Turing machine for the left quotient

      pos := 0,

    -- Rejecting state
