THEOREM: For all natural numbers a, b, and c, if a + b = c, then the sum of the squares of a and b is less than or equal to c squared plus 2ab.

PROOF SKETCH:
1. Start with the given equation a + b = c.
2. Square both sides to obtain (a + b)^2 = c^2.
3. Expand the left side to get a^2 + 2ab + b^2 = c^2.
4. Subtract 2ab from both sides to yield a^2 + b^2 = c^2 - 2ab.
5. Rearrange the inequality to show a^2 + b^2 ≤ c^2 + 2ab, which is trivially true since -2ab ≤ 2ab for all natural numbers a, b.

THEOREM: For all natural numbers a, b, and c, if a ≤ b, then a + c ≤ b + c.

PROOF SKETCH:
1. **Assume** a ≤ b, which by definition means there exists a natural number k such that a + k = b.
2. **Add** c to both sides of the equation a + k = b, resulting in a + k + c = b + c.
3. **Rearrange** the left side to group a and c: (a + c) + k = b + c.
4. **Conclude** that a + c ≤ b + c by the definition of ≤, since (a + c) + k = b + c implies a + c ≤ b + c.

**Natural Language:**
THEOREM: The composition of a polynomial-time function with a polynomial-time decidable language is itself a polynomial-time decidable language, specifically, if \( f \) is a polynomial-time function and \( L \) is a polynomial-time decidable language, then the language \( \{ x \mid f(x) \in L \} \) is also polynomial-time decidable.

PROOF SKETCH:
1. **Definition Recall**: A polynomial-time function \( f \) computes its output in time polynomial in the input size. A polynomial-time decidable language \( L \) has a decider that runs in polynomial time.
2. **Construct Decider for Composed Language**: To decide if \( x \) is in the composed language, compute \( f(x) \) and then check if \( f(x) \) is in \( L \).
3. **Time Complexity Analysis**: The time to compute \( f(x) \) is polynomial in \( |x| \), and the time to decide \( f(x) \in L \) is polynomial in \( |f(x)| \). Since \( |f(x)| \) is polynomially bounded by \( |x| \), the total time is polynomial in \( |x| \).

LEAN HINT:
- Imports: `import complexity.basic`
- Tactics: `apply`, `simp`, `exact` with polynomial time definitions and composition lemmas.

DIFFICULTY: medium

**Lean 4 Proof:**
```lean
import Mathlib
import complexity.basic

open complexity

theorem composition_polynomial_time_decidable
  {α β : Type} [SizeOf α] [SizeOf β]
  (f : α → β) (L : Set β)
  (h_f : poly_time f)
  (h_L : poly_time_decidable L) :
  poly_time_decidable {x | f x ∈ L} :=
begin
  obtain ⟨c₁, h_f_le⟩ := h_f,
  obtain ⟨c₂, h_L_dec⟩ := h_L,
  let c := max c₁ c₂,
  let g : α → Bool := λ x, decide (f x ∈ L),
  have h_g_le : ∀ x, sizeOf (g x) ≤ c * sizeOf x,
  { intro x,
    apply le_trans (h_L_dec (f x)),
    apply mul_le_mul_left,
    apply le_of_lt,
    apply h_f_le,
  },
  have h_g_poly : poly_time g,
  { apply exists.intro c,
    apply h_g_le,
  },
  apply exists.intro c,
  intro x,
  apply decide_eq_decide,
  apply h_L_dec,
end
```
