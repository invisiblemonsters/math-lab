**Natural Language:**
THEOREM: For all natural numbers n, the sum of the first n odd natural numbers is equal to n².

PROOF SKETCH:
1. **Base Case**: Verify the statement for n = 1, where the sum of the first odd number (1) equals 1².
2. **Inductive Step**: Assume the statement holds for some natural number k (i.e., the sum of the first k odd numbers is k²). Then, show it holds for k + 1 by adding the next odd number (2k + 1) to both sides of the inductive hypothesis and simplifying.

LEAN HINT:
```lean
import data.nat.basic
import algebra

theorem sum_first_n_odd (n : ℕ) : (finset.range n).sum (λ i, 2*i + 1) = n² :=
begin
  induction n with k ih,
  { simp [finset.sum_empty] }, -- Base case
  { rw [finset.sum_range_succ, ih], -- Inductive step
    simp [nat.succ_eq_add_one, add_left, add_comm] }
end
```

DIFFICULTY: easy

**Natural Language:**
THEOREM: For all natural numbers a, b, and c, if a + b = c, then the sum of the squares of a and b is less than or equal to c squared plus 2ab.

PROOF SKETCH:
1. Start with the given equation a + b = c.
2. Square both sides to obtain (a + b)^2 = c^2.
3. Expand the left side to get a^2 + 2ab + b^2 = c^2.
4. Subtract 2ab from both sides to yield a^2 + b^2 = c^2 - 2ab.
5. Rearrange the inequality to show a^2 + b^2 ≤ c^2 + 2ab, which is trivially true since -2ab ≤ 2ab for all natural numbers a, b.

LEAN HINT:
- Imports: import algebra
- Tactics: intro, rw, ring, linarith

DIFFICULTY: easy

**Lean 4 Proof:**
```lean
import Mathlib

theorem sum_of_squares_inequality (a b c : Nat) (h : a + b = c) : a^2 + b^2 ≤ c^2 + 2*a*b :=
  by simp [h, Nat.pow]
    rw [Nat.pow, Nat.pow]
    linarith
```
