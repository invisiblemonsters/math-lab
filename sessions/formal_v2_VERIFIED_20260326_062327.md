**Natural Language:**
THEOREM: For all natural numbers n and m, the length of the list obtained by appending two lists of lengths n and m is equal to n + m.

PROOF SKETCH: 
We can prove this by induction on the structure of lists. For the base case, appending two empty lists results in an empty list, and 0 + 0 = 0. For the inductive step, consider two lists, one of length n and the other of length m. Appending them will result in a list where the first element is followed by the rest of the first list and then the entire second list. By the definition of list length, this will be 1 + (n - 1 + m) = n + m, which simplifies to the desired result.

LEAN HINT:
- Imports: `import Data.List`
- Tactics: `induction`, `simp [List.length_append]`, `norm_num`

DIFFICULTY: easy

**Lean 4 Proof:**
```lean
import Mathlib
import Data.List

theorem length_append (α : Type) (as bs : List α) : 
  (as ++ bs).length = as.length + bs.length := 
by induction as 
  case nil => simp [List.length_append, List.length_nil]
  case cons a as hi => simp [List.length_append, List.length_cons] at *
    exact Nat.succ_eq_add_one (hi)
```

PROOF SKETCH: This theorem can be proven using basic properties of list lengths and the definition of the append operation. Specifically, we can use induction on the lengths of the lists or directly apply the existing lemma about list append in Mathlib, which states that the length of the concatenation of two lists is the sum of their lengths.
