THEOREM: The union of two disjoint finite sets A and B is equal to the Cartesian product of the set containing the union of A and B with the set containing the empty set, i.e., A ∪ B = (A ∪ B) × {∅}.

PROOF SKETCH: 
1. Recall the definition of set union: `A ∪ B = {x | x ∈ A ∨ x ∈ B}`.
2. Recall the definition of Cartesian product: `A × B = {(a, b) | a ∈ A ∧ b ∈ B}`.
3. Since A and B are disjoint, every element in `A ∪ B` can be uniquely associated with a pair `(x, ∅)` in `(A ∪ B) × {∅}`.
4. Construct a bijection between `A ∪ B` and `(A ∪ B) × {∅}` to establish set equality.

## Turn 3 — Verified ✓

**Natural Language:**
THEOREM: For any two finite sets A and B, the cardinality of their union is equal to the sum of their cardinalities minus the cardinality of their intersection.

PROOF SKETCH:
1. Recall the definition of set union and intersection.
2. Use the principle of inclusion-exclusion to express |A ∪ B| as |A| + |B| - |A ∩ B|.
3. Since A and B are finite, their cardinalities are well-defined natural numbers.
4. Apply basic arithmetic to show the equality holds.

LEAN HINT:
- Imports: `import data.set.finite`
- Tactics: `simp [set.card_union, set.card_inter]`, `omega` or `linarith` for arithmetic

DIFFICULTY: easy

**Lean 4 Proof:**
```lean
import Mathlib
import data.set.finite

theorem cardinality_union (A B : Set ℕ) [Finite A] [Finite B] : 
  (A ∪ B).toFinite.card = A.toFinite.card + B.toFinite.card - (A ∩ B).toFinite.card := 
by simp [Set.toFinite_card, Set.card_union, Set.card_inter]; omega
```
