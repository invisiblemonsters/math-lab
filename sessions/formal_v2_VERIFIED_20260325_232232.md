PROOF SKETCH:

## Turn 4 — Verified ✓

**Natural Language:**
THEOREM: The union of two finite sets A and B is equal to the union of B and A, and the cardinality of their union is symmetric with respect to the order of the sets.

PROOF SKETCH:
1. **Definition of Union**: Start with the definition of set union, which is commutative by definition in set theory (i.e., A ∪ B = B ∪ A).
2. **Cardinality**: Use the property that the cardinality of a finite set is well-defined and does not depend on the order of elements or the order of union operation.
3. **Symmetry Argument**: Show that the number of elements in A ∪ B is the same as in B ∪ A by applying the formula |A ∪ B| = |A| + |B| - |A ∩ B|, which is symmetric in A and B.

LEAN HINT:
- Imports: `import data.set.basic`
- Tactics: `simp [set.union_comm], rw [set.card_union]`

DIFFICULTY: easy

**Lean 4 Proof:**
```lean
import Mathlib
import data.set.basic

variable {α : Type}

theorem union_comm_and_card_symm (A B : Set α) [Fintype A] [Fintype B] [Fintype (A ∪ B)] :
  A ∪ B = B ∪ A ∧ (A ∪ B).toFinset.card = (B ∪ A).toFinset.card :=
begin
  apply And.intro,
  { exact Set.union_comm _ _, },
  { have h : (A ∪ B).toFinset.card = A.toFinset.card + B.toFinset.card - (A ∩ B).toFinset.card,
    { exact Set.toFinset_card _, },
  have h' : (B ∪ A).toFinset.card = B.toFinset.card + A.toFinset.card - (B ∩ A).toFinset.card,
    { exact Set.toFinset_card _, },
  simp [h, h', Set.inter_comm],
  omega, }
end
```
