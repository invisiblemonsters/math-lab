Specifically, if f: Σ* → Σ* is a polynomial-time function and p: ℕ → ℕ is a polynomial, then the function g defined by g(x) = f(x) concatenated with a string of length p(|x|) is also a polynomial-time function.

PROOF SKETCH:
1. **Polynomial-time function definition**: Recall that a function f is polynomial-time if there exists a polynomial q such that for all inputs x, f(x) can be computed in time q(|x|).
2. **Concatenation operation**: The concatenation of f(x) with a string of length p(|x|) can be computed by first computing f(x) and then generating the additional string of length p(|x|).
3. **Time complexity analysis**: The total time to compute g(x) is the time to compute f(x) plus the time to generate the string of length p(|x|). Since generating a string of length p(|x|) takes time polynomial in |x| (specifically, O(p(|x|)) steps), and f(x) is computed in time q(|x|), the total time is bounded by q(|x|) + p(|x|), which is still a polynomial in |x|.
4. **Conclusion**: Therefore, g is a polynomial-time function.

**Natural Language:**
THEOREM: The composition of two polynomial-time functions is itself a polynomial-time function. Specifically, if f: Σ* → Σ* and g: Σ* → Σ* are polynomial-time functions, then the function h: Σ* → Σ* defined by h(x) = f(g(x)) is also a polynomial-time function.

PROOF SKETCH:
1. **Polynomial-time definition**: Recall that a function f is polynomial-time if there exists a polynomial p such that for all inputs x, the time complexity of f(x) is bounded by p(|x|), where |x| denotes the length of x.
2. **Compose time complexities**: Given f and g are polynomial-time, let p_f and p_g be their respective bounding polynomials. The time complexity of h(x) = f(g(x)) involves first computing g(x) in time ≤ p_g(|x|), then computing f(g(x)) in time ≤ p_f(|g(x)|).
3. **Bound g(x) output length**: Since g is polynomial-time, the length of g(x) is bounded by some polynomial q_g(|x|) (e.g., |g(x)| ≤ q_g(|x|), since output length can't exceed computation time for a reasonable model).
4. **Substitute into f's polynomial**: The time for f(g(x)) is ≤ p_f(q_g(|x|)), which is a polynomial in |x| because the composition of polynomials is a polynomial.
5. **Conclude h is polynomial-time**: Thus, the total time for h(x) is ≤ p_g(|x|) + p_f(q_g(|x|)), which is bounded by a polynomial in |x|.
