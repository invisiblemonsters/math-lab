THEOREM: For any natural numbers a, b, and c, if a ≤ b, then a * c ≤ b * c.

PROOF SKETCH:
1. **Start with the given inequality**: Assume a ≤ b.
2. **Multiply both sides by c**: Since multiplication is monotonic over natural numbers, multiplying both sides of the inequality a ≤ b by c preserves the inequality, yielding a * c ≤ b * c.

LEAN HINT:
- Imports: `import Data.Nat.Basic`
- Tactics: `intro`, `apply Nat.mul_le_mul_right`, `assumption`

DIFFICULTY: easy

**Lean 4 Proof:**
```lean

import Data.Nat.Basic

  by apply Nat.mul_le_mul_right; assumption
```

THEOREM: For all natural numbers n and m, if n ≤ m, then the length of the list [0, 1, ..., n] is less than or equal to the length of the list [0, 1, ..., m].

PROOF SKETCH:
1. **List Length Definition**: Recall that the length of a list in Lean is defined as the number of elements it contains.
2. **Inequality Hypothesis**: Given n ≤ m, we need to relate the lengths of the lists generated by iterating from 0 to n and 0 to m.
3. **List Construction**: The list [0, 1, ..., n] can be constructed using a recursive function or a list comprehension, which inherently depends on the value of n.
4. **Inductive Reasoning**: Use induction on n and m, considering the base case where n = 0 and m = 0, and then the inductive step where assuming the property holds for some n and m, it should hold for n+1 and m+1, leveraging the given inequality.
5. **Application of Nat.le_trans**: Utilize the transitivity of the less-than-or-equal-to relation on natural numbers to establish the desired inequality on list lengths.

LEAN HINT:
```lean
import Data.List
import Data.Nat

theorem list_length_le {n m : ℕ} (h : n ≤ m) : length (List.range (n + 1)) ≤ length (List.range (m + 1)) :=
begin
  -- Apply list.length_range, which states that the length of range k is k
  -- Then use the given hypothesis h and apply Nat.le_trans
  simp [List.range, List.length_range],
  exact h,
end
```

3. **Apply** the cancellation property of natural numbers to show a + c ≤ b + c.

```lean
import Data.Nat.Basic

theorem nat.add_le_add_right (a b c : ℕ) (h : a ≤ b) : a + c ≤ b + c :=
```
