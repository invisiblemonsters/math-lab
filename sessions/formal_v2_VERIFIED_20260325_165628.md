2. **Definition of NP**: A language L is in NP if there exists a polynomial-time decidable relation R and a polynomial p such that x ∈ L ⇨ ∃c (|c| ≤ p(|x|) ∧ R(x, c)).
3. **Construct NP Verifier for L₁**: Given the reducibility function f, we construct a verifier for L₁ using the verifier for L₂. Specifically, for x ∈ L₁, the certificate c' for L₁ can be the same as the certificate c for f(x) in L₂, adjusted by the polynomial-time function f.
4. **Polynomial Bounds**: Show that the length of c' is polynomially bounded by |x|, using the polynomial bounds of f and L₂'s verifier.
5. **Decidability of the Relation**: Demonstrate that the relation R₁ for L₁, defined as R₁(x, c') = R₂(f(x), c'), is polynomial-time decidable because R₂ is and f is polynomial-time computable.

LEAN HINT:
```lean
import Complexity.Reductions
import Complexity.NP

theorem poly_red_closed_NP {L₁ L₂ : Language} (h_red : L₁ ≤ₚ L₂) (h_np : L₂ ∈ NP) : L₁ ∈ NP :=
begin
  -- Unfold definitions
  obtain ⟨f, f_poly, f_red⟩ := h_red,
  obtain ⟨R, R_poly, R_np⟩ := h_np,
  -- Construct the NP verifier for L₁
  use (λx c, R (f x) c),
  -- Prove the relation is polynomial-time decidable
  exact (R_poly.comp f_poly),
  -- Verify the NP condition
  intros x,
  split,
  { intro hx, obtain ⟨c, hc₁, hc₂⟩ := h_red.forward hx, use c, split, exact hc₁, exact hc₂ },
  { intro h, apply h_red.backward, exact h }
end
```
