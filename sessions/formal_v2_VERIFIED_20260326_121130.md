**Natural Language:**
THEOREM: The sum of the elements of a list of natural numbers is equal to the sum of the elements of the list obtained by removing duplicates from the original list, if the original list contains no duplicates.

PROOF SKETCH:
1. **Base Case**: Show that an empty list has a sum of 0, and removing duplicates from an empty list leaves it unchanged.
2. **Inductive Step**: Assume the property holds for all lists of length n. For a list of length n+1, consider two cases:
   a. The new element is already in the list. Removing duplicates will not affect the sum since the element was already counted.
   b. The new element is not in the list. Removing duplicates leaves the list unchanged, and the sum is updated by adding the new element.
3. **Conclusion**: By induction, the sum remains unchanged when removing duplicates from a list with no duplicates.

LEAN HINT:
- Imports: `import Data.List`
- Tactics: `induction`, `cases`, `simp`, `linarith`

import Mathlib

theorem sum_remove_duplicates (l : List ℕ) (h : l.Nodup) : l.sum = (l.dedup).sum := 
by induction l with
| nil => simp [List.sum, List.dedup]
| cons x xs ih => 
  cases h with
  | cons hd tl =>
    simp [List.sum, List.dedup]
    cases List.mem_decidable x xs with
    | isTrue h' => 
      have : (x :: xs).dedup = xs.dedup := by simp [List.dedup, h']
      simp [this, ih tl]
    | isFalse h' => 
      have : (x :: xs).dedup = x :: xs.dedup := by simp [List.dedup, h']
      simp [this, ih tl, List.sum]
      linarith

## Turn 3 — Verified ✓

**Natural Language:**
THEOREM: For any two lists of natural numbers, the sum of their concatenation is equal to the sum of the first list plus the sum of the second list.

PROOF SKETCH: 
1. Define two arbitrary lists `A` and `B` of natural numbers.
2. Use induction on the structure of `A`.
   - Base case: If `A` is empty, the concatenation `A ++ B` is just `B`, so the sum is `0 + sum B`, which simplifies to `sum B`.
   - Inductive step: Assume the property holds for a list `A`. Consider `A` with an additional element `a`. The sum of `(a :: A) ++ B` is `a + sum (A ++ B)`. By the inductive hypothesis, this is `a + (sum A + sum B)`, which rearranges to `(a + sum A) + sum B`, proving the property for the concatenated list.
3. Conclude by mathematical induction that the sum of the concatenation of two lists is the sum of their individual sums.

LEAN HINT: 
- Imports: `import Data.List`
- Tactics: `induction A`, `simp [List.sum_cons, add_assoc]`, `exact`

DIFFICULTY: easy

**Lean 4 Proof:**
```lean
import Mathlib
import Data.List

theorem sum_concat (A B : List Nat) : (A ++ B).sum = A.sum + B.sum :=
  by induction A
     case nil => simp [List.sum_nil, List.nil_append]
     case cons a A ih => simp [List.sum_cons, List.cons_append]
                          exact add_assoc a A.sum B.sum ▸ ih
```
