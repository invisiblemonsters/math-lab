**Natural Language:**
THEOREM: The image of a finite set of natural numbers under a function is also finite.

PROOF SKETCH:
1. **Definition Recall**: A set is finite if it has a finite number of elements.
2. **Function Application**: Consider a function f: ℕ → ℕ and a finite set S = {a₁, a₂, ..., aₙ}.
3. **Image Construction**: The image f(S) = {f(a₁), f(a₂), ..., f(aₙ)}.
4. **Finiteness Preservation**: Since S has n elements, f(S) has at most n elements (due to potential duplicates), which is still finite.

LEAN HINT:
- Imports: `import data.set.finite`
- Tactics: `intro`, `apply`, `simp`, `exact` (using `set.finite.image` from Mathlib)

DIFFICULTY: easy

**Lean 4 Proof:**
```lean
import Mathlib
import data.set.finite

theorem finite_image (f : ℕ → ℕ) (S : Set ℕ) (h : S.Finite) : (f '' S).Finite :=
  by exact Set.Finite.image h f
```
