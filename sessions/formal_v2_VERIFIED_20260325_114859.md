**Natural Language:**
THEOREM: The union of two finite subsets of the natural numbers is also a finite subset of the natural numbers.

PROOF SKETCH:
1. Let A and B be finite subsets of ℕ.
2. By definition, there exist natural numbers n and m such that A ⊆ {0, 1, ..., n} and B ⊆ {0, 1, ..., m}.
3. The union A ∪ B is then a subset of {0, 1, ..., max(n, m)}.
4. Since {0, 1, ..., max(n, m)} is finite, A ∪ B is finite.

LEAN HINT:
```lean
import data.nat.basic

theorem finite_union {A B : set ℕ} (hA : finite A) (hB : finite B) : finite (A ∪ B) :=
begin
  -- Use the definition of finite sets in Mathlib
  cases hA with n hA, cases hB with m hB,
  -- Construct the bounding set for the union
  use max n m,
  -- Show the union is bounded by max(n, m)
  intros x hx,
  cases hx; [apply hA, apply hB]; assumption
end
```

DIFFICULTY: easy

**Natural Language:**
THEOREM: The composition of two polynomial-time computable functions is also polynomial-time computable. Specifically, if f: Σ* → Σ* and g: Σ* → Σ* are polynomial-time computable functions, then the composition h(x) = f(g(x)) is also polynomial-time computable.

PROOF SKETCH:
1. Recall the definition of polynomial-time computable functions: a function f is polynomial-time computable if there exists a polynomial p such that for all inputs x, the computation time of f(x) is bounded by p(|x|), where |x| denotes the length of x.
2. Let p_f and p_g be the polynomials bounding the computation time of f and g, respectively.
3. The computation time of h(x) = f(g(x)) is the sum of the time to compute g(x) and the time to compute f(g(x)).
4. Since |g(x)| ≤ p_g(|x|) (as g is polynomial-time computable), the time to compute f(g(x)) is bounded by p_f(p_g(|x|)).
5. Therefore, the total computation time of h(x) is bounded by p_g(|x|) + p_f(p_g(|x|)), which is a polynomial in |x|.

LEAN HINT:
- Imports: `import Complexity.Basic`
- Tactics: `apply`, `use` with polynomial bounds, `simp` for algebraic manipulation of polynomials.

DIFFICULTY: medium

**Lean 4 Proof:**
```lean
import Mathlib
import Complexity.Basic

theorem composition_polynomial_time
  {f g : String → String}
  (hf : ∃ p, ∀ x, f.computationTime x ≤ p x.length)
  (hg : ∃ p, ∀ x, g.computationTime x ≤ p x.length) :
  ∃ p, ∀ x, (f ∘ g).computationTime x ≤ p x.length :=
begin
  rcases hf with ⟨pf, hf⟩,
  rcases hg with ⟨pg, hg⟩,
  use fun x => pg x + pf (pg x),
  intro x,
  apply le_trans _ (add_le_add (hg x) (hf (g x))),
  apply le_of_eq,
  simp [(∘), computationTime]
end
```
