**Natural Language:**
THEOREM: The union of two finite subsets of the natural numbers is also a finite subset of the natural numbers.

PROOF SKETCH:
1. Let A and B be finite subsets of ℕ.
2. By definition, there exist natural numbers n and m such that A ⊆ {0, 1, ..., n} and B ⊆ {0, 1, ..., m}.
3. The union A ∪ B is then a subset of {0, 1, ..., max(n, m)}.
4. Since {0, 1, ..., max(n, m)} is finite, A ∪ B is finite.

LEAN HINT:
```lean
import data.nat.basic

theorem finite_union {A B : set ℕ} (hA : finite A) (hB : finite B) : finite (A ∪ B) :=
begin
  -- Use the definition of finite sets in Mathlib
  cases hA with n hA, cases hB with m hB,
  -- Construct the bounding set for the union
  use max n m,
  -- Show the union is bounded by max(n, m)
  intros x hx,
  cases hx; [apply hA, apply hB]; assumption
end
```

DIFFICULTY: easy

THEOREM: The image of a finite set under a function is also a finite set. Specifically, if A is a finite subset of X and f: X → Y is a function, then f(A) = { f(x) | x ∈ A } is a finite subset of Y.

PROOF SKETCH:
1. Recall the definition of a finite set: a set that has the same cardinality as some natural number n.
2. Use the fact that the image of a set under a function can be constructed by iterating over each element in the domain set and applying the function.
3. Since A is finite, we can list its elements as a₁, a₂, ..., aₙ.
4. The image f(A) will then consist of the elements f(a₁), f(a₂), ..., f(aₙ), which is a finite list (possibly with duplicates).
5. Apply the "to_set" operation to this list to form the image set, which remains finite because it is bounded by the length of the original list.

LEAN HINT:
- Imports: import data.set.finite
- Tactics: apply finite.image, exact finite.of_fintype (if needed for specific fintype instances)

DIFFICULTY: easy
