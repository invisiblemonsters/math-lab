THEOREM: The composition of two polynomial-time functions is also a polynomial-time function. Specifically, if f: Σ* → Σ* is polynomial-time computable and g: Σ* → Σ* is polynomial-time computable, then the function h: Σ* → Σ* defined by h(x) = g(f(x)) is also polynomial-time computable.

2. **Derive Bound for h**: For any input x, the time complexity of h(x) = g(f(x)) is bounded by q(p(|x|)), which is a polynomial in |x|.
3. **Formalize Bound**: Show that q(p(|x|)) is indeed a polynomial in |x|, ensuring h is polynomial-time computable.

- Imports: `import Complexity.Basic`

import Complexity.Basic

theorem polynomial_time_composition 
  (f g : String → String) 
  (p q : ℕ → ℕ) 
  (hf : ∀ x, f x ≠ none ∧ (f x).length ≤ p x.length) 
  (hg : ∀ x, g x ≠ none ∧ (g x).length ≤ q x.length) 
  (poly_p : ∃ a b : ℕ, ∀ x, p x ≤ a * x + b) 
  (poly_q : ∃ c d : ℕ, ∀ x, q x ≤ c * x + d) :
  ∃ r : ℕ → ℕ, (∃ e f : ℕ, ∀ x, r x ≤ e * x + f) ∧ 
  ∀ x, (g ∘ f) x ≠ none ∧ ((g ∘ f) x).length ≤ r x.length :=
begin
  use (λ x => q (p x)),
  split,
  { obtain ⟨a, b, h_p⟩ := poly_p,
    obtain ⟨c, d, h_q⟩ := poly_q,
    use [a * c, c * b + d],
    intro x,
    apply le_trans (h_q (p x)),
    apply le_trans (mul_le_mul h_p (le_refl c)),
    ring },
  { intro x,
    have h_fx := hf x,
    have h_gfx := hg (f x),
    simp [Function.comp],
    split,
    { exact h_gfx.1 },
    { apply le_trans h_gfx.2,
      apply h_q } }
end
