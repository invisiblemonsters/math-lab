2. **Use NP characterization**: Since L₂ ∈ NP, there exists a polynomial p and a decidable predicate R such that y ∈ L₂ ⇨ ∃c (|c| ≤ p(|y|) ∧ R(y, c)).
3. **Construct NP certificate for L₁**: For x ∈ L₁, consider the certificate c' = c for f(x). We need to show that x ∈ L₁ ⇨ ∃c' (|c'| ≤ p'(|x|) ∧ R'(x, c')), where p' is a polynomial derived from p and f's running time.
4. **Verify polynomial bounds**: Since f is polynomial-time, |f(x)| ≤ q(|x|) for some polynomial q. Then, |c'| ≤ p(|f(x)|) ≤ p(q(|x|)), which is a polynomial in |x|.
5. **Decidability of R'**: Define R'(x, c') = R(f(x), c'). Since R is decidable and f is polynomial-time computable, R' is decidable.

- Import `complexity.basic`, `computability.primrec`, and `data.nat.basic`
- Use `exists.intro` to construct the certificate, `apply` known lemmas about polynomial composition, and `aesop` for decidability closure.

import Mathlib
import complexity.basic
import computability.primrec
import data.nat.basic

open complexity computability

theorem reduction_preserves_np (L₁ L₂ : Language) (h : L₁ ≤ₚ L₂) (hL₂ : L₂ ∈ NP) : L₁ ∈ NP :=
begin
  cases h with f hf,
  cases hL₂ with p R hR,
  let p' := λ x, p (hf.2 x) + hf.2 x,
  let R' := λ x c', R (f x) c',
  apply exists.intro p',
  apply exists.intro R',
  intro x,
  constructor,
  { intro hx,
    cases hR (f x) (hf.1 hx) with c hc,
    apply exists.intro c,
    constructor,
    { apply le_trans _ (p (hf.2 x)),
      { exact hc.1 },
      { apply le_add_left },
    },
    { exact hc.2 } },
  { intro hc',
    cases hc' with c' hc',
    apply hf.1,
    apply hR (f x),
    apply exists.intro c',
    exact hc' },
  { intro x c',
    apply aesop,
    apply hR.decidable },
end
