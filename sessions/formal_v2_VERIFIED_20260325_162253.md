THEOREM: The composition of a polynomial-time function with itself (i.e., f ∘ f) is also a polynomial-time function. Specifically, if f: Σ* → Σ* is polynomial-time computable, then the function f ∘ f: Σ* → Σ* defined by (f ∘ f)(x) = f(f(x)) is also polynomial-time computable.

PROOF SKETCH: 
1. **Definition Recall**: A function f is polynomial-time computable if there exists a polynomial p such that for all inputs x, the computation time of f(x) is bounded by p(|x|), where |x| denotes the length of x.
2. **Composing Functions**: Given f is polynomial-time computable, we need to show that f ∘ f is also polynomial-time computable. 
3. **Time Complexity Analysis**: The time to compute f(f(x)) is the time to compute f(x) plus the time to compute f(f(x)). Since f is polynomial-time, there exists a polynomial p such that the time to compute f(x) is at most p(|x|). Then, the time to compute f(f(x)) is at most p(|f(x)|). 
4. **Polynomial Bound**: We need to show that p(|f(x)|) is bounded by another polynomial in |x|. Since |f(x)| ≤ p(|x|) (because the output length of a polynomial-time function is polynomially bounded), we can substitute to get p(p(|x|)), which is still a polynomial in |x| because the composition of polynomials is a polynomial.
5. **Conclusion**: Therefore, the total time to compute f ∘ f is bounded by p(|x|) + p(p(|x|)), which is a polynomial in |x|, proving that f ∘ f is polynomial-time computable.
