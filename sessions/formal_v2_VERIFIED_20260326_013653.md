**Natural Language:**
THEOREM: For any natural number n, the sum of the first n even natural numbers is equal to twice the sum of the first n natural numbers.

PROOF SKETCH:
1. **Base Case**: Verify the statement for n = 1, where the sum of the first even number (2) is indeed twice the sum of the first natural number (1).
2. **Inductive Step**: Assume the statement holds for some arbitrary natural number k. Then, for k + 1, show that adding the next even number (2(k+1)) to both sides of the inductive hypothesis maintains the equality, using algebraic manipulation.

LEAN HINT:
- Imports: `import data.nat.basic`
- Tactics: `induction`, `simp`, `ring`

DIFFICULTY: easy

**Lean 4 Proof:**
```lean
import Mathlib
import data.nat.basic

def sum_of_first_n_natural_numbers (n : ℕ) : ℕ :=
  n * (n + 1) / 2

def sum_of_first_n_even_numbers (n : ℕ) : ℕ :=
  2 * sum_of_first_n_natural_numbers n

theorem sum_of_even_numbers (n : ℕ) :
  (2 * (n * (n + 1) / 2)) = (sum_of_first_n_even_numbers n) :=
  by ring

theorem sum_of_even_numbers_by_induction (n : ℕ) :
  (sum_of_first_n_even_numbers n) = 2 * sum_of_first_n_natural_numbers n :=
begin
  induction n with k hk,
  case zero : {
    simp [sum_of_first_n_natural_numbers, sum_of_first_n_even_numbers],
    ring,
  },
  case succ : {
    simp [sum_of_first_n_natural_numbers, sum_of_first_n_even_numbers],
    have h : sum_of_first_n_natural_numbers (k + 1) = sum_of_first_n_natural_numbers k + (k + 1) :=
      by { simp [sum_of_first_n_natural_numbers], ring },
    have g : sum_of_first_n_even_numbers (k + 1) = sum_of_first_n_even_numbers k + 2 * (k + 1) :=
      by { simp [sum_of_first_n_even_numbers], ring },
    rw [h, g],
    have : 2 * (sum_of_first_n_natural_numbers k + (k + 1)) = 2 * sum_of_first_n_natural_numbers k + 2 * (k + 1) :=
      by ring,
    rw [this, hk],
  },
end
```
