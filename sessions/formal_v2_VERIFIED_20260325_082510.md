THEOREM: The image of a finite set under a function is also finite.

PROOF SKETCH:

2. **Function Application**: Consider a function `f: A → B` and a finite set `S ⊆ A`.
3. **Image Definition**: The image of `S` under `f` is defined as `f(S) = {b ∈ B | ∃a ∈ S, f(a) = b}`.
4. **Finite Image Argument**: Since `S` is finite, we can list its elements as `a₁, a₂, ..., aₙ`. The image `f(S)` can have at most `n` distinct elements (since each `aᵢ` maps to some `bᵢ`, but some `bᵢ` might be repeated).

**Natural Language:**
THEOREM: The intersection of two finite sets of natural numbers is also finite.

PROOF SKETCH:
1. **Definition Recall**: A set of natural numbers is finite if it has a finite number of elements.
2. **Intersection Property**: For any two sets A and B, the intersection A ∩ B is a subset of both A and B.
3. **Subset of Finite Set**: Since A is finite and A ∩ B is a subset of A, then A ∩ B must also be finite (as subsets of finite sets are finite).

LEAN HINT:
- Imports: `import data.set.finite`
- Tactics: `intro`, `apply`, `subset_of_finite` (or similar finite set properties from Mathlib)

DIFFICULTY: easy

**Lean 4 Proof:**

```lean
theorem finite_intersection (A B : Set ℕ) (hA : Set.Finite A) : Set.Finite (A ∩ B) :=
```

**Natural Language:**
THEOREM: The composition of a polynomial function with a polynomial-time computable function yields a polynomial-time computable function.

PROOF SKETCH:
1. **Definitions**: Recall that a polynomial function `p: ℕ → ℕ` is of the form `p(n) = a_k n^k + ... + a_1 n + a_0` with non-negative coefficients, and a polynomial-time computable function `f: ℕ → ℕ` satisfies `f(n) ≤ p(n)` for some polynomial `p`.
2. **Composition**: Consider the composition `f ∘ p`, i.e., `(f ∘ p)(n) = f(p(n))`.
3. **Polynomial Bound**: Show that `f(p(n))` is bounded by a polynomial in `n`. Since `p(n)` is a polynomial in `n` and `f(m) ≤ q(m)` for some polynomial `q`, then `f(p(n)) ≤ q(p(n))`. 
4. **Polynomial Closure**: Demonstrate that the composition `q ∘ p` is still a polynomial in `n` (as the composition of two polynomials is a polynomial).
5. **Conclusion**: Conclude that `f ∘ p` is polynomial-time computable, as it is bounded by the polynomial `q ∘ p`.

```lean
theorem poly_time_compose_poly {f : ℕ → ℕ} {p : ℕ → ℕ} 
  (hf : poly_time f) (hp : is_polynomial p) :
  poly_time (λ n, f (p n)) :=
begin
  -- Unfold definitions and use polynomial composition properties
  unfold poly_time at *,
  -- Extract polynomial bounds for f and p
  cases hf with q hf,
  cases hp with k hp,
  -- Construct the composed polynomial
  use [q ∘ p],
  -- Show the composed function is bounded by the composed polynomial
  intro n,
  apply hf,
```

```lean
import Mathlib
import Complexity.Basic

-- Non-negative coefficients polynomial
def is_polynomial (p : ℕ → ℕ) : Prop :=
  ∃ (k : ℕ) (a : Fin (k + 1) → ℕ), ∀ (n : ℕ),
    p n = ∑ i in Fin.range (k + 1), a i * n ^ (k - i)

-- Polynomial-time computable function
def poly_time (f : ℕ → ℕ) : Prop :=
  ∃ (p : ℕ → ℕ), is_polynomial p ∧ ∀ (n : ℕ), f n ≤ p n

theorem poly_time_compose_poly {f : ℕ → ℕ} {p : ℕ → ℕ} (hf : poly_time f) (hp : is_polynomial p) :
  poly_time (λ n, f (p n)) :=
begin
  -- Unfold definitions and use polynomial composition properties
  cases hf with q hf,
  cases hp with k hp,
  cases hf with q_le hf,
  -- Extract coefficients of p
  cases hp with a hp,

  -- Construct the composed polynomial
  let r : ℕ → ℕ := λ n, ∑ i in Fin.range (k + 1), a i * (∑ j in Fin.range (k + 1), q_le * (a j * n ^ (k - j))) ^ (k - i),

  -- Show the composed function is bounded by the composed polynomial
  use r,
  split,
  { -- Show r is a polynomial
    use 2 * k,
    let b : Fin (2 * k + 1) → ℕ := λ i, if i < k then a (Fin.castLe (Nat.le_of_lt (Fin.is_lt i)) i) else 0,
    use b,
    intro n,
    simp [r],
    -- Expand r
    apply Fin.sum_congr,
    intro i,
    simp [b],
    -- Simplify coefficients
    split_ifs,
    { simp [hp, Fin.castLe, Nat.le_of_lt],
      -- Simplify the power
      apply Fin.sum_congr,
      intro j,
      simp [b],
      split_ifs,
      { simp [hp, Fin.castLe, Nat.le_of_lt],
        ring, },
      { simp [Fin.is_lt, Nat.not_lt], }, },
    { simp [Fin.is_lt, Nat.not_lt], }, },
  { -- Show f ∘ p is bounded by r
    intro n,
    apply hf,
    -- Simplify the composition
    simp [r],
    -- Bound f
    apply q_le,
    -- Simplify the polynomial
    apply Fin.sum_congr,
    intro i,
    -- Simplify coefficients
    simp [hp],
    ring, },
end
```
