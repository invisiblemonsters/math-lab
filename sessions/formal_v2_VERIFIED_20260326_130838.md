**Natural Language:**
THEOREM: The sum of the first n natural numbers, denoted as 1 + 2 + ... + n, is equal to n(n + 1)/2.

PROOF SKETCH:
1. **Base Case**: Verify the formula for n = 1, where 1 = 1(1 + 1)/2.
2. **Inductive Step**: Assume the formula holds for some arbitrary natural number k, i.e., 1 + 2 + ... + k = k(k + 1)/2. Then, show it holds for k + 1 by adding (k + 1) to both sides and simplifying to (k + 1)(k + 2)/2.

LEAN HINT:
- Imports: `import data.nat.basic`
- Tactics: `induction n`, `simp [nat.add_comm, nat.mul_assoc, nat.add_left_comm]`, `rw add_comm`

DIFFICULTY: easy

**Lean 4 Proof:**
```lean
import Mathlib
import data.nat.basic

theorem sum_of_first_n_natural_numbers (n : ℕ) : 
  (List.range (n + 1)).foldl (· + ·) 0 = n * (n + 1) / 2 :=
by induction n
   case zero => simp
   case succ k IH => 
     have : (List.range (k + 2)).foldl (· + ·) 0 = 
             (List.range (k + 1)).foldl (· + ·) 0 + (k + 1) := 
     by simp [List.range_succ, List.foldl_append]
     rw [this, IH]
     ring
```
