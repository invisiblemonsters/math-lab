**Natural Language:**
THEOREM: The composition of a polynomial-time function with a polynomial-time decidable language results in a polynomial-time decidable language. Formally, if f: Σ* → Σ* is a polynomial-time function and L ⊆ Σ* is a polynomial-time decidable language, then the language { w | f(w) ∈ L } is also polynomial-time decidable.

PROOF SKETCH: 
1. **Decidability of L**: Since L is polynomial-time decidable, there exists a Turing machine M_L that decides L in polynomial time.
2. **Polynomial-time function f**: The function f can be computed by a Turing machine M_f in polynomial time.
3. **Composition**: Construct a Turing machine M that, on input w, computes f(w) using M_f and then checks if the result is in L using M_L. 
4. **Polynomial-time bound**: The running time of M is the sum of the running times of M_f and M_L, both of which are polynomial in the input size |w|. Since the composition of polynomials is a polynomial, the overall running time of M is polynomial in |w|.

- Imports: `import complexity.basic`
- Tactics: `apply`, `use`, `simp`, `ring` for polynomial composition
- Key definitions: `polynomial_time`, `decidable_language`, `turing_machine`

  (f : Σ* → Σ*)
  (L : set Σ*)
  (h_f : polynomial_time f)
  (h_L : decidable_language L) :
  decidable_language { w | f w ∈ L } :=
begin
  obtain ⟨M_L, h_M_L⟩ := h_L,
  obtain ⟨M_f, h_M_f⟩ := h_f,
  let M := M_f.seq (M_L.adapt (λ _, ())),
  apply decidable_language.mk,
  use M,
  intro w,
  simp only [turing_machine.seq_accepts, turing_machine.adapt_accepts],
  have : M_f.eval w = some (f w) := h_M_f w,
  simp only [this, option.mem_def],
  apply h_M_L,
end

    ∀ (w : Σ*), turing_machine.eval_time (composition_polynomial_time_decidable f L h_f h_L).some w ≤ p w.length :=
