THEOREM: The image of a finite set under a function is also finite.

PROOF SKETCH:
1. **Definition of Finite Sets**: Recall that a set is finite if there exists a natural number n such that the set has exactly n elements.
2. **Function Application**: Consider a function f: A → B and a finite set S ⊆ A. We need to show that f(S) = { f(x) | x ∈ S } is finite.
3. **Induction on Cardinality**: Use induction on the number of elements in S. If S has 0 elements, f(S) is empty (finite). For the inductive step, assume all sets with n elements have finite images. For a set with n+1 elements, remove one element, apply the induction hypothesis, and then add the image of the removed element (which is a singleton, hence finite) to the finite image.
4. **Union of Finite Sets**: The image of the set with n+1 elements is the union of the image of the first n elements (finite by induction hypothesis) and the image of the remaining element (a singleton, finite). The union of two finite sets is finite (by previously verified theorem).

- Key definitions: `set.finite`, `set.image`

THEOREM: The class of polynomial-time decidable languages is closed under the operation of Kleene star (i.e., if a language L is in P, then the language L* = { w | w = u1 || u2 || ... || uk, where k ≥ 0 and each ui ∈ L } is also in P).

PROOF SKETCH: 
To prove closure under Kleene star, we can construct a polynomial-time Turing machine that decides L*. The machine will check all possible decompositions of the input string w into a concatenation of strings u1, u2, ..., uk, each of which must be in L. Since L is in P, each check can be done in polynomial time. However, the naive approach of checking all possible decompositions would lead to exponential time complexity due to the number of possible splits. Instead, we use dynamic programming to efficiently verify the decomposition: for each position i in the string, we precompute whether the substring from the start to i can be formed by concatenating strings from L. This approach ensures that the overall time complexity remains polynomial.

THEOREM: The class of decidable languages is closed under the operation of intersection, i.e., if languages L1 and L2 are decidable, then the language L1 ∩ L2 = {w | w ∈ L1 ∧ w ∈ L2} is also decidable.

PROOF SKETCH:
To prove closure under intersection, we construct a decider for L1 ∩ L2 using the deciders for L1 and L2. Given an input string w, the decider for L1 ∩ L2 first checks if w is in L1 using the decider for L1. If w is not in L1, it immediately rejects. If w is in L1, it then checks if w is in L2 using the decider for L2, returning the result. Since both deciders for L1 and L2 halt on all inputs, the constructed decider also halts, ensuring decidability of L1 ∩ L2.
