**Natural Language:**
THEOREM: The image of a finite set under a function is also finite.

PROOF SKETCH:
1. **Definition of Finite Sets**: Recall that a set is finite if there exists a natural number n such that the set has exactly n elements.
2. **Function Application**: Consider a function f: A → B and a finite set S ⊆ A. We need to show that f(S) = { f(x) | x ∈ S } is finite.
3. **Induction on Cardinality**: Use induction on the number of elements in S. If S has 0 elements, f(S) is empty (finite). For the inductive step, assume all sets with n elements have finite images. For a set with n+1 elements, remove one element, apply the induction hypothesis, and then add the image of the removed element (which is a singleton, hence finite) to the finite image.
4. **Union of Finite Sets**: The image of the set with n+1 elements is the union of the image of the first n elements (finite by induction hypothesis) and the image of the remaining element (a singleton, finite). The union of two finite sets is finite (by previously verified theorem).

LEAN HINT:
- Imports: `import data.set.finite`
- Tactics: `induction`, `apply`, `simp`, `exact`
- Key definitions: `set.finite`, `set.image`

DIFFICULTY: medium

**Lean 4 Proof:**
```lean
import Mathlib
import data.set.finite

theorem image_of_finite_set_is_finite {α β : Type} (f : α → β) (S : Set α) (hS : S.Finite) : (f '' S).Finite :=
  induction hS using Set.Finite.induction_on
  case empty => exact Set.Finite.empty
  case insert a s ha hs ih =>
    have : f '' (insert a s) = insert (f a) (f '' s) := by ext; simp
    have : (f '' s).Finite := ih
    exact Set.Finite.insert _ this
Note: `Set.Finite.induction_on` is used instead of regular induction to directly utilize the induction principle for finite sets.
```

**Natural Language:**
THEOREM: The Cartesian product of two finite subsets of natural numbers is finite.

PROOF SKETCH:
1. **Definition of Finite Sets**: Recall that a set of natural numbers is finite if it has a cardinality of some natural number n.
2. **Cartesian Product Definition**: The Cartesian product A × B of sets A and B is the set of all ordered pairs (a, b) where a ∈ A and b ∈ B.
3. **Cardinality Argument**: If |A| = m and |B| = n, then |A × B| = m * n, which is also a natural number.
4. **Closure Under Multiplication**: Since the product of two natural numbers is natural, the Cartesian product of finite sets is finite.

LEAN HINT:
- Imports: `import data.set.finite`
- Tactics: `simp`, `omega`, `apply` with `set.finite.prod` (if available in Mathlib), otherwise construct the proof using `finite.of_surjective` or similar.

**Natural Language:**
THEOREM: The union of a finite set of natural numbers with a singleton set containing a natural number is also finite.

PROOF SKETCH:
1. **Definition of Finite Sets**: Recall that a set is finite if it has a specific, countable number of elements (using the definition from Mathlib, e.g., `finite A ↔ ∃ n, A ⊆ {0, 1, ..., n}`).
2. **Union with Singleton**: Consider a finite set `A` and a singleton set `{a}` where `a` is a natural number. We need to show `A ∪ {a}` is finite.
3. **Case Analysis**: If `a ∈ A`, then `A ∪ {a} = A`, which is finite by assumption. If `a ∉ A`, then `A ∪ {a}` has exactly one more element than `A`, which is still finite because the union of two finite sets (here, `A` and `{a}`) is finite by a previously verified theorem (Theorem 2).

LEAN HINT:
- Import necessary modules: `import data.set.finite`
- Use tactics: `cases` for case analysis, `apply` with the previously verified theorem on the union of finite sets, `simp` for set membership simplifications.

DIFFICULTY: easy

**Lean 4 Proof:**
```lean
import Mathlib

theorem union_singleton_finite (A : Set ℕ) (a : ℕ) (hA : A.Finite) : (A ∪ {a}).Finite := 
  if ha : a ∈ A then 
    by have : A ∪ {a} = A := Set.union_eq_left_iff.mpr ha; 
       rw this; 
       exact hA 
  else 
    by apply Set.finite.union hA; 
       exact Set.finite_singleton a
```
