**Natural Language:**
THEOREM: The set of decidable languages is closed under the right quotient operation. Formally, if L₁ and L₂ are decidable languages over an alphabet Σ, then the language L₁ / L₂ = { x ∈ Σ* | ∃ y ∈ L₂, xy ∈ L₁ } is also decidable.

PROOF SKETCH:
1. **Definition Recall**: A language L is decidable if there exists a Turing machine (or an algorithm) that halts and correctly decides membership in L for all inputs.
2. **Construct Decider for L₁ / L₂**: Given input x, iterate over all possible strings y in Σ* (up to a length that ensures xy is within the length bounds where L₁'s decidability is effective). For each y, check if y ∈ L₂ and xy ∈ L₁ using the deciders for L₂ and L₁, respectively.
3. **Termination Argument**: Since L₂ is decidable, for any x, there are only finitely many y's of length up to |x| + k (for some constant k) that need to be checked, ensuring the algorithm terminates.
4. **Correctness**: If such a y exists, the algorithm will find it and accept x; otherwise, it will reject x after exhausting all possibilities.

LEAN HINT:
- Imports: `import computability.language`
- Tactics: `apply`, `use` (for existential witness), `decidable.em`, `simp`, `induction` on string length
- Define the decider function explicitly using `λ x, ...` and use `rfind` or `bounded_search` from `computability` module

import Mathlib
import computability.language

theorem decidable_right_quotient {Σ : Type} [fintype Σ] (L₁ L₂ : language Σ) 
  (d₁ : decidable_pred L₁) (d₂ : decidable_pred L₂) : decidable_pred (L₁ / L₂) :=
begin
  let decider := λ x, 
    let search_bound := x.length + (d₁.decidable x).some.length in
    let candidates := (list_of_finset (finset.bounded_finset search_bound Σ)).map (λ y, y.mk_string) in
    (candidates.rfind (λ y, d₂ y ∧ d₁ (y ++ x))).is_some,

  apply decidable_pred.mk,
  intro x,
  use decider x,
  split,
  { intro h,
    obtain ⟨y, hy, hxy⟩ := h,
    have := (d₂.decidable y).some_spec,
    have := (d₁.decidable (y ++ x)).some_spec,
    simp only [decider],
    apply rfind_spec,
    use y.mk_string,
    simp only [hy, hxy, this, and_self],
    apply list.mem_map_of_mem,
    apply list_of_finset_mem,
    apply finset.mem_bounded_finset,
    apply le_trans _ (add_le_add_right (mk_string_length_le y _) _),
    exact nat.le_add_right _ _ },
  { intro h,
    obtain ⟨y, hy, hxy⟩ := rfind_spec h,
    use y,
    split,
    { exact d₂ y hy },
    { exact d₁ (y ++ x) hxy } }
end

THEOREM: Polynomial-time reductions preserve the property of being decidable. Formally, if a language L₁ is polynomial-time reducible to a language L₂, and L₂ is decidable, then L₁ is also decidable.
