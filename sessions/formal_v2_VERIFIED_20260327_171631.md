2. **Composition**: Consider a function f: Σ* → Σ* that is polynomial-time computable and a predicate P: Σ* → Bool that is polynomial-time verifiable.
3. **Verification**: To verify the composition P(f(x)), we need to check if P holds for the output of f(x). Since f is polynomial-time computable, it can be evaluated in polynomial time. 4. **Polynomial Bound**: Show that the total verification time is bounded by a polynomial in the size of x, using the fact that the size of f(x) is polynomially bounded by the size of x (from the polynomial-time computability of f).

- Imports: `import Complexity.Basic`
- Tactics: `apply`, `simp`, `exact`, `omega` (for polynomial bounds)

variable (P : Σ* → Bool)

theorem composition_polynomial_time_verifiable
  (hf : polynomial_time_computable f)
