THEOREM: For all natural numbers n and m, if n ≤ m, then n + 1 ≤ m + 1.

PROOF SKETCH: 
1. **Start with the given inequality**: Assume n ≤ m.
2. **Apply the definition of ≤ for natural numbers**: Use the fact that n ≤ m means there exists a k such that n + k = m.
3. **Substitute and simplify**: Add 1 to both sides of the equation n + k = m, yielding (n + 1) + k = m + 1.
4. **Conclude the inequality**: This shows that n + 1 ≤ m + 1 by the definition of ≤.

- Imports: `import data.nat.basic`
