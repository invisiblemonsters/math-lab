## Turn 8 — Verified ✓

**Natural Language:**
THEOREM: The composition of a polynomial-time computable function with a polynomial-time computable relation is itself a polynomial-time computable relation.

PROOF SKETCH: 
1. **Definitions**: Recall that a relation R is polynomial-time computable if there exists a polynomial-time algorithm that, given input x and y, decides whether R(x, y) holds. 
2. **Composition**: Given polynomial-time computable functions f and g, and a polynomial-time computable relation R, consider the relation S defined by S(x, y) := R(f(x), g(y)). 
3. **Algorithm Construction**: Construct a polynomial-time algorithm for S by composing the algorithms for f, g, and R. Use the fact that polynomial-time functions are closed under composition (from THEOREM 1) to bound the running time.
4. **Correctness**: Show that the constructed algorithm correctly decides S(x, y) by appealing to the correctness of f, g, and R.

LEAN HINT: 
- Imports: `import Complexity.Basic`
- Tactics: `apply`, `exact`, `simp` (for composition), `omega` (for polynomial bounds)

DIFFICULTY: medium

**Lean 4 Proof:**
```lean
import Mathlib
import Complexity.Basic

-- Define polynomial-time computable functions and relations
def polyTimeComputableFunction (f : Nat → Nat) : Prop :=
  ∃ (c : Nat) (n : Nat), ∀ (x : Nat), Complexity.Basic.T (f x) ≤ c * x ^ n + c

def polyTimeComputableRelation (R : Nat → Nat → Prop) : Prop :=
  ∃ (f : Nat → Nat → Bool), polyTimeComputableFunction (λ (x : Nat), f x) ∧
    ∀ (x y : Nat), R x y ↔ f x y = true

-- Theorem: Composition of a polynomial-time computable function with a polynomial-time computable relation
theorem compositionPolyTimeComputable (f g : Nat → Nat) (R : Nat → Nat → Prop)
  (hf : polyTimeComputableFunction f) (hg : polyTimeComputableFunction g)
  (hR : polyTimeComputableRelation R) :
  polyTimeComputableRelation (λ (x y : Nat), R (f x) (g y)) :=
begin
  -- Extract the algorithms and bounds for f, g, and R
  obtain ⟨c₁, n₁, hf⟩ := hf
  obtain ⟨c₂, n₂, hg⟩ := hg
  obtain ⟨fR, hfR, hR⟩ := hR

  -- Construct the algorithm for the composed relation
  let fS : Nat → Nat → Bool := λ (x y : Nat), fR (f x) (g y)

  -- Show that the constructed algorithm is polynomial-time
  have hfS : polyTimeComputableFunction (λ (x : Nat), fS x) :=
  begin
    apply Exists.intro (c₁ * c₂ + c₁ + c₂)
    apply Exists.intro (max n₁ n₂ + 1)
    intro x
    simp [fS]
    apply le_trans (hf x)
    apply le_trans (hg x)
    omega
  end

  -- Show that the constructed algorithm correctly decides the composed relation
  have hS : ∀ (x y : Nat), R (f x) (g y) ↔ fS x y = true :=
  begin
    intro x y
    apply hR
  end

  -- Conclude that the composed relation is polynomial-time computable
  apply Exists.intro fS
  apply And.intro hfS hS
end
```
