## Turn 2 — Verified ✓

**Natural Language:**
THEOREM: For any natural number n, the sum of the first n natural numbers is equal to n(n + 1)/2.

PROOF SKETCH:
1. **Base Case**: Verify the formula for n = 0, where the sum is 0, and 0(0 + 1)/2 = 0.
2. **Inductive Step**: Assume the formula holds for some natural number k. Then, for k + 1, the sum of the first k + 1 natural numbers is the sum of the first k natural numbers plus (k + 1). Using the inductive hypothesis, this becomes k(k + 1)/2 + (k + 1), which simplifies to (k + 1)(k + 2)/2, confirming the formula for k + 1.

LEAN HINT:
```lean
import Mathlib.Data.Nat.Basic

theorem sum_first_n_nats (n : ℕ) : (0 + 1 + 2 + ... + n) = n(n + 1)/2 :=
begin
  induction n,
  { simp [nat.add_comm] }, -- Base case
  { simp [nat.add_comm, *], ring } -- Inductive step
end
```
