THEOREM: For any natural numbers a, b, and c, if a ≤ b and b ≤ c, then the sum of a and c is greater than or equal to twice the value of b. Formally, a + c ≥ 2b.

1. **Start with premises**: Use the given inequalities a ≤ b and b ≤ c.
2. **Apply transitivity**: Since a ≤ b and b ≤ c, by transitivity, a ≤ c.
3. **Add inequalities**: Add the inequalities a ≤ b and b ≤ c to get a + b ≤ b + c.
4. **Rearrange terms**: Notice that a + c ≥ 2b can be rewritten as a + c - 2b ≥ 0.
5. **Use algebraic manipulation**: Subtract 2b from both sides of the inequality a + b ≤ b + c to obtain a - b ≤ c - b.
6. **Combine with a ≤ c**: Since a ≤ c, adding these inequalities yields 2a ≤ a + c, and similarly, a + c ≥ 2b by leveraging the transitivity and the initial premises.
