**Natural Language:**
THEOREM: The composition of a polynomial-time decidable function with another polynomial-time decidable function results in a polynomial-time decidable function. Specifically, if f: ℕ → ℕ and g: ℕ → ℕ are polynomial-time decidable functions, then the function h: ℕ → ℕ defined by h(x) = f(g(x)) is also polynomial-time decidable.

PROOF SKETCH:
1. **Definition of Polynomial-Time Decidable**: Recall that a function f is polynomial-time decidable if there exists a polynomial p such that for all inputs x, f(x) can be computed in time bounded by p(|x|), where |x| denotes the length of x.
2. **Composition of Functions**: Given h(x) = f(g(x)), we need to show that h is polynomial-time decidable.
3. **Polynomial Bound**: Let p_f be the polynomial bound for f and p_g be the polynomial bound for g. We need to construct a polynomial p_h that bounds the computation time of h.
4. **Constructing p_h**: The time to compute g(x) is bounded by p_g(|x|), and the time to compute f(g(x)) is bounded by p_f(|g(x)|). Since |g(x)| ≤ p_g(|x|) (as g is polynomial-time decidable), we have p_f(|g(x)|) ≤ p_f(p_g(|x|)). Thus, the total time for h(x) is bounded by p_g(|x|) + p_f(p_g(|x|)), which is a polynomial in |x|.
5. **Conclusion**: Therefore, h is polynomial-time decidable.

- Imports: `import Data.Polynomial` and relevant complexity theory definitions.

def polyTimeDecidable (f : ℕ → ℕ) : Prop :=
  ∃ p : ℕ[X], ∀ x, (f x).time ≤ p.eval (x.length)

-- Assume f and g are polynomial-time decidable
theorem composition_is_poly_time_decidable
  (f g : ℕ → ℕ) (hf : polyTimeDecidable f) (hg : polyTimeDecidable g) :
  polyTimeDecidable (f ∘ g) :=
begin
  -- Extract the polynomial bounds for f and g
  obtain ⟨p_f, hf⟩ := hf
  obtain ⟨p_g, hg⟩ := hg

  -- Construct the polynomial bound for h
  let p_h := p_g + p_f.comp p_g

  -- Show that p_h bounds the computation time of h
  use p_h
  intro x

  -- Bound the time to compute g(x)
  have hg_bound : (g x).time ≤ p_g.eval (x.length) := hg x

  -- Bound the time to compute f(g(x))
  have hf_bound : (f (g x)).time ≤ p_f.eval ((g x).length) := hf (g x)

  -- Use the fact that |g(x)| ≤ p_g(|x|)
  have g_len_bound : (g x).length ≤ p_g.eval (x.length) := hg_bound

  -- Combine the bounds
  have hf_bound' : (f (g x)).time ≤ p_f.eval (p_g.eval (x.length)) :=
    hf_bound.trans (p_f.mono g_len_bound)

  -- Conclude that p_h bounds the computation time of h
  have h_bound : (f ∘ g) x.time ≤ p_h.eval (x.length) :=
    (hg_bound.trans (add_le_add_right hf_bound' _))
