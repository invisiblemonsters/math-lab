**Natural Language:**
THEOREM: For all natural numbers a, b, and c, if a ≤ b, then a + c ≤ b + c.

PROOF SKETCH:
1. **Assume** a ≤ b, which by definition means there exists a natural number d such that a + d = b.
2. **Add** c to both sides of the equation a + d = b, yielding (a + c) + d = b + c.
3. **Apply** the definition of ≤ to conclude a + c ≤ b + c.

LEAN HINT:
```lean
import Data.Nat.Basic

theorem add_le_add_right {a b c : ℕ} (h : a ≤ b) : a + c ≤ b + c :=
begin
  induction h,
  case nat.le_refl : { refl },
  case nat.le_succ : _ _ h_le { exact nat.le_succ (a + c) }
end
```
