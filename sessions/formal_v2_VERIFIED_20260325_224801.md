**Natural Language:**
THEOREM: For any finite set A, the union of A with the empty set is equal to A itself.

PROOF SKETCH: 
1. Recall the definition of set union: `A ∪ B = {x | x ∈ A ∨ x ∈ B}`.
2. Substitute the empty set for B: `A ∪ ∅ = {x | x ∈ A ∨ x ∈ ∅}`.
3. Since `x ∈ ∅` is always false, the union simplifies to `{x | x ∈ A}`, which is just A.
4. Use the axiom of extensionality to conclude equality.

LEAN HINT:
```lean
import Data.Set.Basic

theorem union_empty (A : Set) : A ∪ ∅ = A := by
  simp [Set.union, Set.empty]
```

DIFFICULTY: easy

**Lean 4 Proof:**

  Set.ext (fun x => by simp [Set.mem_union, Set.mem_empty])

**Natural Language:**
THEOREM: For any finite set A, the intersection of A with itself is equal to A itself.

PROOF SKETCH:
1. Recall the definition of set intersection: A ∩ B = {x | x ∈ A ∧ x ∈ B}.
2. Substitute B with A: A ∩ A = {x | x ∈ A ∧ x ∈ A}.
3. Simplify the condition using the idempotent law of conjunction: x ∈ A ∧ x ∈ A ⇔ x ∈ A.
4. Conclude that A ∩ A = {x | x ∈ A} = A.

LEAN HINT:
- Imports: import set_theory.set.basic
- Tactics: simp, refl, ext (for set extensionality)

DIFFICULTY: easy

**Lean 4 Proof:**
```lean
import Mathlib

theorem intersection_idempotent {α : Type} (A : Set α) : A ∩ A = A := 
  by ext x; simp [Set.mem_inter]; tauto
```
