PROOF SKETCH:

3. **Time Complexity**: The time to compute g(x) is bounded by p_g(|x|). Then, the time to compute f(g(x)) is bounded by p_f(|g(x)|). Since |g(x)| ≤ p_g(|x|) (as the output size of a polynomial-time function is polynomially bounded), we have p_f(p_g(|x|)), which is a polynomial in |x|.
4. **Polynomial Product**: The resulting polynomial is the composition p_f ∘ p_g, which is a polynomial (specifically, the product of the degrees of p_f and p_g plus the sum of their coefficients, but more accurately, the composition of polynomials is a polynomial).

**Natural Language:**
THEOREM: If a language L is in P, then its complement (the set of all strings not in L) is also in P.

PROOF SKETCH:
1. **Definition of P**: Recall that a language L is in P if there exists a polynomial-time decidable algorithm (e.g., a Turing machine) that decides membership in L.
2. **Complement Construction**: To decide the complement of L, denoted ¬L, we can use the same polynomial-time algorithm for L and simply invert its output (accept when the original algorithm rejects, and vice versa).
3. **Polynomial-Time Preservation**: Since the original algorithm runs in polynomial time, inverting its output does not increase the running time; thus, the algorithm for ¬L also runs in polynomial time.
4. **Closure Conclusion**: Therefore, if L is in P, then ¬L is also in P, demonstrating that P is closed under complementation.

theorem P_closed_under_complement {L : Language} (hL : L ∈ P) : ¬L ∈ P :=
begin
  -- Unfold the definition of P for L
  obtain ⟨M, p, hM⟩ := hL,
  -- Construct the machine for ¬L by inverting M's output
  let M' := M.complement,
  -- Verify M' runs in the same polynomial time p
  use M',
  use p,
  -- Show that M' decides ¬L in polynomial time
  intros x,
  -- Invert the decidability of M on x
  simp [M'.decides_iff],
  -- Apply hM to ensure M's correctness

end
