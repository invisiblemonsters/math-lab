**Natural Language:**
THEOREM: For any finite set A, the union of A with the empty set is equal to A itself.

PROOF SKETCH: 
1. Recall the definition of set union: `A ∪ B = {x | x ∈ A ∨ x ∈ B}`.
2. Substitute the empty set for B: `A ∪ ∅ = {x | x ∈ A ∨ x ∈ ∅}`.
3. Since `x ∈ ∅` is always false, the union simplifies to `{x | x ∈ A}`, which is just A.
4. Use the axiom of extensionality to conclude equality.

LEAN HINT:
```lean

theorem union_empty (A : Set) : A ∪ ∅ = A := by
  simp [Set.union, Set.empty]
```
