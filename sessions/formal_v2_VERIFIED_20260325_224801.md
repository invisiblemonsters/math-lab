PROOF SKETCH:

3. **Time Complexity**: The time to compute g(x) is bounded by p_g(|x|). Then, the time to compute f(g(x)) is bounded by p_f(|g(x)|). Since |g(x)| ≤ p_g(|x|) (as the output size of a polynomial-time function is polynomially bounded), we have p_f(p_g(|x|)), which is a polynomial in |x|.
4. **Polynomial Product**: The resulting polynomial is the composition p_f ∘ p_g, which is a polynomial (specifically, the product of the degrees of p_f and p_g plus the sum of their coefficients, but more accurately, the composition of polynomials is a polynomial).

3. **Polynomial-Time Composition**: Use the previously verified theorem that the composition of polynomial-time computable functions is polynomial-time computable (Theorem 8) to show that h is polynomial-time computable.
4. **Preserve Membership**: Demonstrate that h preserves membership, i.e., x ∈ L₁ if and only if h(x) ∈ L₃, by applying the preservation properties of the individual reductions f and g.

**Lean 4 Proof:**
```lean
import Complexity.Reductions

theorem reduction_composition {L₁ L₂ L₃ : Language} (f : L₁ ≤ₚ L₂) (g : L₂ ≤ₚ L₃) : L₁ ≤ₚ L₃ := by {
  -- Construct the composition function h = g ∘ f
  let h := λ x, g.fn (f.fn x),
  
  -- Extract the polynomial-time computability of f and g
  have f_poly := f.poly_time,
  have g_poly := g.poly_time,
  
  -- Verify h is polynomial-time computable using Theorem 8
  have h_poly := poly_time_composition f_poly g_poly,
  
  -- Prove preservation of membership using f and g's properties
  have preservation : ∀ x, x ∈ L₁ ↔ h x ∈ L₃ := by {
    intro x,
    have f_preserve := f.preserve x,
    have g_preserve := g.preserve (f.fn x),
    exact f_preserve.trans g_preserve
  },
  
  -- Construct the reduction from L₁ to L₃
  exact ⟨h, h_poly, preservation⟩
}
```
