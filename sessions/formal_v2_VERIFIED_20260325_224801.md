**Natural Language:**
THEOREM: For any finite set A, the union of A with the empty set is equal to A itself.

PROOF SKETCH: 
1. Recall the definition of set union: `A ∪ B = {x | x ∈ A ∨ x ∈ B}`.
2. Substitute the empty set for B: `A ∪ ∅ = {x | x ∈ A ∨ x ∈ ∅}`.
3. Since `x ∈ ∅` is always false, the union simplifies to `{x | x ∈ A}`, which is just A.
4. Use the axiom of extensionality to conclude equality.

LEAN HINT:

  simp [Set.union, Set.empty]

**Natural Language:**
THEOREM: The composition of two polynomial-time computable functions is itself a polynomial-time computable function, and the resulting polynomial is the product of the individual polynomials.

PROOF SKETCH:
1. **Definitions**: Recall that a function f is polynomial-time computable if there exists a polynomial p such that f(x) can be computed in time p(|x|), where |x| denotes the length of input x.
2. **Composition**: Consider two polynomial-time computable functions f and g with polynomials p_f and p_g, respectively. We need to show that the composition h(x) = f(g(x)) is polynomial-time computable.
3. **Time Complexity**: The time to compute g(x) is bounded by p_g(|x|). Then, the time to compute f(g(x)) is bounded by p_f(|g(x)|). Since |g(x)| ≤ p_g(|x|) (as the output size of a polynomial-time function is polynomially bounded), we have p_f(p_g(|x|)), which is a polynomial in |x|.
4. **Polynomial Product**: The resulting polynomial is the composition p_f ∘ p_g, which is a polynomial (specifically, the product of the degrees of p_f and p_g plus the sum of their coefficients, but more accurately, the composition of polynomials is a polynomial).

LEAN HINT:
- Import necessary modules: `import Data.Polynomial` and `import Complexity.Basic`
- Define polynomial-time computable functions using existing structures (e.g., `PolynomialTimeComputable`).
- Use `apply` with `composition_polynomial` to establish the polynomial bound.
- Tactics: `simp`, `apply`, `exact`, `ring` for polynomial manipulations.

DIFFICULTY: Medium

import Data.Polynomial

open Polynomial

structure PolynomialTimeComputable (f : ℕ → ℕ) (p : Polynomial ℕ) where
  bound : ∀ x, f x ≤ p (x.natSize)

noncomputable def natSize (x : ℕ) : ℕ := x

theorem composition_polynomial_time_computable (f g : ℕ → ℕ) (p_f p_g : Polynomial ℕ)
  (h_f : PolynomialTimeComputable f p_f) (h_g : PolynomialTimeComputable g p_g) :
  PolynomialTimeComputable (f ∘ g) (p_f ∘ p_g) :=
begin
  apply PolynomialTimeComputable.mk,
  intro x,
  have g_bound : g x ≤ p_g (x.natSize) := h_g.bound x,
  have f_bound : f (g x) ≤ p_f ((g x).natSize) := h_f.bound (g x),
  have size_bound : (g x).natSize ≤ p_g (x.natSize) := by omega,
  have final_bound : f (g x) ≤ p_f (p_g (x.natSize)) :=
    by apply le_trans f_bound; apply p_f.mono; exact size_bound,
  exact final_bound,
end

theorem composition_polynomial_is_product (p_f p_g : Polynomial ℕ) :
  (p_f ∘ p_g).degree = p_f.degree * p_g.degree :=
by simp [Polynomial.degree_comp]

theorem composition_polynomial_coefficients (p_f p_g : Polynomial ℕ) :
  (p_f ∘ p_g).coeffs = p_f.coeffs.map (fun c => c * p_g.coeffs.prod) :=
by simp [Polynomial.coeffs_comp]
