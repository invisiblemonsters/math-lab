**Natural Language:**
THEOREM: The Cartesian product of two finite subsets of the natural numbers is also a finite subset of the natural numbers.

PROOF SKETCH:
1. Let A and B be finite subsets of ℕ.
2. By definition, there exist natural numbers m and n such that A has exactly m elements and B has exactly n elements.
3. The Cartesian product A × B consists of all ordered pairs (a, b) where a ∈ A and b ∈ B.
4. Each element of A can pair with each element of B, resulting in m × n possible pairs.
5. Since m × n is a natural number, A × B has a finite number of elements.
6. Therefore, A × B is a finite subset of ℕ × ℕ, which is a subset of ℕ (using a pairing function encoding).

LEAN HINT:
- Imports: `import data.set.finite`
- Tactics: `apply set.finite.prod`, `simp`, `exact`
- Consider using Mathlib's `set.finite` namespace for existing finite set properties.

DIFFICULTY: easy

**Lean 4 Proof:**

```lean
import data.set.finite

theorem cartesian_product_finite (A B : Set ℕ) (hA : Set.Finite A) (hB : Set.Finite B) : 
    Set.Finite (Set.prod A B) := 
by apply Set.Finite.prod; exact ‹Set.Finite A›; exact ‹Set.Finite B›
```

**Natural Language:**
THEOREM: The image of a finite subset of the natural numbers under a polynomial function is also a finite subset of the natural numbers.

PROOF SKETCH:
1. **Definition Recall**: A polynomial function p: ℕ → ℕ has the form p(n) = a_k n^k + ... + a_1 n + a_0, where coefficients are natural numbers.
2. **Finite Image Argument**: For a finite set S = {s_1, ..., s_m}, the image p(S) = {p(s_1), ..., p(s_m)} is finite because it has at most m distinct elements (by function application).
3. **Polynomial Evaluation**: Since polynomial evaluation is well-defined for all natural numbers, each p(s_i) is a natural number, ensuring p(S) ⊆ ℕ.
4. **Conclusion**: Thus, p(S) is finite as it's a finite collection of natural numbers.

LEAN HINT:
- Imports: `import Data.Nat.Polynomial`
- Tactics: `induction` on the finite set, `apply` definition of polynomial function, `simp` to evaluate image, `exact` for set finiteness.

DIFFICULTY: medium

**Lean 4 Proof:**
```lean
import Mathlib
import Data.Nat.Polynomial

open Finset

theorem finite_image_of_polynomial (p : ℕ → ℕ) (S : Finset ℕ) :
  (S.image p).Finite :=
begin
  apply finite.image,
  exact S.finite_toSet,
  exact p,
end

theorem polynomial_map_nat (p : Polynomial ℕ) (n : ℕ) : 
  Polynomial.eval n p ∈ ℕ :=
begin
  induction p using Polynomial.induction_on,
  case h_add : a b ha hb =>
    simp [Polynomial.eval_add],
    apply Nat.add_closure,
    exact ha,
    exact hb,
  case h_monomial : n a =>
    simp [Polynomial.eval_monomial],
    apply Nat.mul_closure,
    apply Nat.pow_closure,
    exact n,
    exact a,
  case h_C : a =>
    simp [Polynomial.eval_C],
    exact a,
end

theorem finite_image_of_polynomial_map (p : Polynomial ℕ) (S : Finset ℕ) :
  (S.image (Polynomial.eval · p)).Finite :=
begin
  apply finite_image_of_polynomial,
  exact p,
  exact S,
  intro n,
  apply polynomial_map_nat,
  exact p,
  exact n,
end
```
