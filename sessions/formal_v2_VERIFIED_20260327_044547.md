**Natural Language:**
THEOREM: The intersection of two finite sets of natural numbers is also finite.

PROOF SKETCH:
1. **Definition of Finite Set**: Recall that a set is finite if there exists a natural number n such that the set has exactly n elements.
2. **Intersection Property**: The intersection of two sets A and B, denoted A ∩ B, consists of elements common to both A and B.
3. **Cardinality Argument**: If A has m elements and B has n elements, then A ∩ B has at most min(m, n) elements, which is still finite.
4. **Constructive Proof**: Given finite sets A and B, list their elements and filter out those not present in both, resulting in a finite list.

LEAN HINT:
- Imports: `import data.set.finite`
- Tactics: `intro`, `apply`, `exact`, `simp`, `omega`

DIFFICULTY: easy

**Lean 4 Proof:**

```lean
theorem finite_intersection (A B : Set ℕ) (hA : A.Finite) (hB : B.Finite) : (A ∩ B).Finite :=
  Set.Finite.inter hA hB
```

**Natural Language:**
THEOREM: The union of two finite sets of natural numbers is also finite.

PROOF SKETCH: 
1. **Definition of Finite Set**: Recall that a set of natural numbers is finite if it has a cardinality of some natural number n.
2. **Union Cardinality**: Use the fact that the cardinality of the union of two sets is at most the sum of their cardinalities (i.e., |A ∪ B| ≤ |A| + |B|).
3. **Finite Sum**: Since both |A| and |B| are finite, their sum is also finite.
4. **Conclusion**: Therefore, the union set A ∪ B has a finite cardinality, making it finite.

LEAN HINT:
- Imports: `import data.nat.basic` and `import data.set.basic`
- Tactics: `intro`, `apply`, `simp`, `omega` for handling natural number arithmetic

DIFFICULTY: easy

**Lean 4 Proof:**
```lean
import Mathlib

theorem union_of_finite_sets_is_finite {A B : Set ℕ} (hA : Set.Finite A) (hB : Set.Finite B) : 
  Set.Finite (A ∪ B) := 
Set.Finite.union hA hB
```
