**Natural Language:**
THEOREM: The intersection of two finite sets of natural numbers is also finite.

PROOF SKETCH:
1. **Definition of Finite Set**: Recall that a set is finite if there exists a natural number n such that the set has exactly n elements.
2. **Intersection Property**: The intersection of two sets A and B, denoted A ∩ B, consists of elements common to both A and B.
3. **Cardinality Argument**: If A has m elements and B has n elements, then A ∩ B has at most min(m, n) elements, which is still finite.
4. **Constructive Proof**: Given finite sets A and B, list their elements and filter out those not present in both, resulting in a finite list.

LEAN HINT:
- Imports: `import data.set.finite`
- Tactics: `intro`, `apply`, `exact`, `simp`, `omega`

DIFFICULTY: easy

theorem finite_intersection (A B : Set ℕ) (hA : A.Finite) (hB : B.Finite) : (A ∩ B).Finite :=
