**Natural Language:**
THEOREM: For any natural number n, the sum of the first n natural numbers (i.e., 1 + 2 + ... + n) is equal to n(n + 1)/2.

PROOF SKETCH: 
1. Start with an arbitrary natural number n.
2. Use mathematical induction to prove the statement.
3. Base case: For n = 1, show that 1 = 1(1 + 1)/2.
4. Inductive step: Assume the statement holds for some k, i.e., 1 + 2 + ... + k = k(k + 1)/2. Then, show that 1 + 2 + ... + k + (k + 1) = (k + 1)(k + 2)/2 by using the inductive hypothesis and simplifying.

LEAN HINT:
- Imports: `import data.nat.basic`
- Tactics: `induction n`, `simp`, `rw`, `exact`

DIFFICULTY: easy

**Lean 4 Proof:**
```lean
import Mathlib
import data.nat.basic

def sum_of_first_n (n : ℕ) : ℕ := (n * (n + 1)) / 2

theorem sum_of_first_n_eq (n : ℕ) : ∑ i in Finset.range (n + 1), i = sum_of_first_n n :=
by induction n with
| zero => simp [sum_of_first_n, Finset.range, Finset.sum]
| succ k ih => 
  simp [Finset.sum_range_succ]
  rw [ih]
  simp [sum_of_first_n]
  ring
```

import Mathlib
