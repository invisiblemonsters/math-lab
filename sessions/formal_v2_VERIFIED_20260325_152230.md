**Natural Language:**
THEOREM: For all natural numbers n, the sum of the first n odd numbers is equal to n². Specifically, 1 + 3 + 5 + ... + (2n-1) = n².

PROOF SKETCH: We can prove this by mathematical induction. 
- Base case: For n = 1, the sum is 1, which is equal to 1².
- Inductive step: Assume the statement holds for some k, i.e., 1 + 3 + 5 + ... + (2k-1) = k². We need to show it holds for k+1. Adding the next odd number (2(k+1)-1) to both sides gives k² + 2k + 1 = (k+1)², which completes the induction.
