THEOREM: The symmetric difference of two sets is commutative, i.e., for any sets A and B, the symmetric difference A Δ B is equal to B Δ A.

PROOF SKETCH:
1. **Definition Recall**: The symmetric difference A Δ B is defined as (A \ B) ∪ (B \ A).
2. **Commutativity of Union**: Use the previously verified theorem that the union of two disjoint sets is commutative.
3. **Set Difference Commutativity**: Show that A \ B = B \ A is not generally true, but their union with the other set difference is commutative.
4. **Apply Union Commutativity**: Since (A \ B) and (B \ A) are disjoint, apply the commutativity of union to conclude A Δ B = B Δ A.

LEAN HINT:
```lean

theorem symmetric_diff_comm {α : Type*} (A B : set α) : A Δ B = B Δ A :=
by { unfold symmetric_diff, rw union_comm, }
```

**Natural Language:**
THEOREM: For all natural numbers a, b, and c, if a ≤ b, then a + c ≤ b + c.

PROOF SKETCH: 
1. **Start with the given inequality**: Assume a ≤ b.
2. **Apply the definition of ≤ for natural numbers**: This means there exists some natural number d such that a + d = b.
3. **Add c to both sides of the equation**: (a + d) + c = b + c, which simplifies to a + c + d = b + c.
4. **Use the associative property of addition**: This shows that a + c ≤ b + c because adding d to a + c results in b + c.

LEAN HINT:
- Imports: `import data.nat.basic`
- Tactics: `intro`, `apply`, `simp [nat.add_assoc]`

DIFFICULTY: easy

**Lean 4 Proof:**
```lean

theorem add_le_add_right (a b c : Nat) (h : a ≤ b) : a + c ≤ b + c :=
  match h with
  | Nat.le.intro d hd =>
    Nat.le.intro (by simp [← hd, Nat.add_assoc])
```

## Turn 4 — Verified ✓

**Natural Language:**
THEOREM: The composition of a polynomial function with an exponential function results in an exponential function, specifically if f(n) = n^k for some constant k and g(n) = 2^n, then the composition h(n) = f(g(n)) is an exponential function in terms of n.

PROOF SKETCH:
1. **Definitions**: Start with the definitions of polynomial and exponential functions.
2. **Composition**: Substitute g(n) into f(n) to get h(n) = (2^n)^k = 2^(n*k).
3. **Simplify**: Recognize that 2^(n*k) can be rewritten as (2^k)^n, which is of the form a^n where a = 2^k, a constant.
4. **Conclusion**: Conclude that h(n) is an exponential function in n, as it fits the form b^n for some constant b.

LEAN HINT:
- Imports: `import algebra.polynomial` and relevant exponential function definitions.
- Tactics: Use `simp` for simplification, `ring` for algebraic manipulations, and `exact` to apply definitions directly.

DIFFICULTY: Medium

**Lean 4 Proof:**
```lean
import Mathlib
import algebra.polynomial

variable (k n : ℕ)

-- Define the polynomial function f(n) = n^k
def f (n : ℕ) : ℕ := n ^ k

-- Define the exponential function g(n) = 2^n
def g (n : ℕ) : ℕ := 2 ^ n

-- Define the composition h(n) = f(g(n))
def h (n : ℕ) : ℕ := f (g n)

theorem composition_is_exponential : h n = (2 ^ k) ^ n :=
  by simp [h, f, g, ← pow_assoc]

theorem exponential_form : ∃ b : ℕ, h n = b ^ n :=
  by exact ⟨2 ^ k, composition_is_exponential⟩
```
