PROOF SKETCH: 
1. **Start with the given inequality**: Assume a ≤ b.
2. **Apply the definition of ≤ for natural numbers**: This means there exists some natural number d such that a + d = b.
3. **Add c to both sides of the equation**: (a + d) + c = b + c, which simplifies to a + c + d = b + c.
4. **Use the associative property of addition**: This shows that a + c ≤ b + c because adding d to a + c results in b + c.

**Natural Language:**
THEOREM: The composition of a polynomial function with an exponential function results in an exponential function, specifically if f(n) = n^k for some constant k and g(n) = 2^n, then the composition h(n) = f(g(n)) is an exponential function in terms of n.

PROOF SKETCH:
1. **Definitions**: Start with the definitions of polynomial and exponential functions.
2. **Composition**: Substitute g(n) into f(n) to get h(n) = (2^n)^k = 2^(n*k).
3. **Simplify**: Recognize that 2^(n*k) can be rewritten as (2^k)^n, which is of the form a^n where a = 2^k, a constant.
4. **Conclusion**: Conclude that h(n) is an exponential function in n, as it fits the form b^n for some constant b.

LEAN HINT:
- Imports: `import algebra.polynomial` and relevant exponential function definitions.
- Tactics: Use `simp` for simplification, `ring` for algebraic manipulations, and `exact` to apply definitions directly.

DIFFICULTY: Medium

**Lean 4 Proof:**
```lean

import algebra.polynomial

variable (k n : ℕ)

-- Define the polynomial function f(n) = n^k
def f (n : ℕ) : ℕ := n ^ k

-- Define the exponential function g(n) = 2^n
def g (n : ℕ) : ℕ := 2 ^ n

-- Define the composition h(n) = f(g(n))

PROOF SKETCH:
1. **Definitions**: Recall that a polynomial function is of the form p(n) = a_k n^k + ... + a_1 n + a_0, where a_i are coefficients and k is a non-negative integer.
2. **Case Analysis**: Consider two cases based on the degrees of f and g:
   - **Case 1**: k = m. The sum is straightforwardly a polynomial of degree k (by Theorem 3).
   - **Case 2**: k ≠ m. Without loss of generality, assume k > m. The term n^k dominates n^m for sufficiently large n, thus the sum is a polynomial of degree k.
3. **Formalization**: Use the definition of polynomial functions and properties of exponents to show the sum satisfies the polynomial form.

def polynomial_function (f : ℕ → ℕ) := ∃ k : ℕ, ∃ c : ℕ, ∀ n, f n = n^k + c

-- Theorem statement

begin
  -- Extract degrees and coefficients from hypotheses
  cases hf with k_f c_f h_f,
  cases hg with k_g c_g h_g,
  -- Case analysis on degrees
  by_cases h_deg : k_f = k_g,
  { -- Case 1: Equal degrees
    apply polynomial_function.of_degree (k_f) (c_f + c_g) _,
    simp [h_f, h_g, h_deg] },
  { -- Case 2: Unequal degrees
    have h_max : max k_f k_g = k_f ∨ max k_f k_g = k_g,
    { by_cases h : k_f > k_g; simp [h] },
    cases h_max,
    { -- Subcase 2.1: k_f is maximum
      apply polynomial_function.of_degree (k_f) (c_f) _,
      simp [h_f, h_g, h_deg] },
    { -- Subcase 2.2: k_g is maximum
      apply polynomial_function.of_degree (k_g) (c_g) _,
      simp [h_f, h_g, h_deg] } }
end
```

4. **Conclusion**: Therefore, the composition of two polynomial functions is a polynomial function.
