THEOREM: For any list of natural numbers, the sum of the elements of the list is equal to the sum of the elements of the list obtained by removing duplicates, plus the sum of the elements that were removed as duplicates.

PROOF SKETCH:

For the base case, an empty list trivially satisfies the condition. For the inductive step, we assume the property holds for a list of length n, and then consider adding a new element. If the new element is not a duplicate, the sum remains unchanged. If it is a duplicate, we show that the sum of the original list is equal to the sum of the list without the duplicate plus the duplicate element itself. This requires using properties of list sums and the fact that removing duplicates does not affect the sum of the remaining unique elements.

THEOREM: For any two natural numbers a and b, the sum of the elements of the list [a, b] is equal to the sum of the elements of the list [b, a].

PROOF SKETCH: 
To prove this, we can use the basic properties of addition in natural numbers, specifically commutativity. The sum of a list [a, b] is defined as a + b. Similarly, the sum of [b, a] is b + a. By the commutative property of addition, a + b = b + a, thus the sums are equal.
