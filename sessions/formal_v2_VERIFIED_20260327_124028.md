**Natural Language:**
THEOREM: For any natural number n, the sum of the first n natural numbers is equal to n times the average of the first and last term, i.e., 1 + 2 + ... + n = n * (1 + n) / 2.

PROOF SKETCH: 
1. Use mathematical induction on n.
2. Base case (n = 1): Verify that 1 = 1 * (1 + 1) / 2, which simplifies to 1 = 1.
3. Inductive step: Assume the formula holds for some k ∈ ℕ, i.e., 1 + 2 + ... + k = k * (1 + k) / 2.
4. Show that it holds for k + 1, i.e., 1 + 2 + ... + k + (k + 1) = (k + 1) * (1 + (k + 1)) / 2, by manipulating the left-hand side using the inductive hypothesis.

LEAN HINT:
- Imports: `import data.nat.basic`
- Tactics: `induction`, `simp`, `ring`
- Key definitions: `nat`, `+`, `*`, `/`

DIFFICULTY: easy

theorem sum_of_first_n_natural_numbers (n : ℕ) : ∑ i in Finset.range (n + 1), i = n * (n + 1) / 2 :=
  Finset.sum_arithmetic_series n 0 1

**Natural Language:**
THEOREM: For any natural numbers a, b, and c, the product of a and the sum of b and c is equal to the sum of the products of a and b and a and c. Formally, a * (b + c) = a * b + a * c.

PROOF SKETCH: 
1. Start with the left-hand side (LHS) of the equation: a * (b + c).
2. Apply the distributive property of multiplication over addition, which states that multiplying a number by the sum of two numbers is equal to the sum of the products of the number with each of the two numbers.
3. This directly transforms the LHS into a * b + a * c, which is the right-hand side (RHS) of the equation.
4. Conclude that the LHS equals the RHS, thus proving the theorem.

LEAN HINT:
- Imports: `import Algebra`
- Tactics: `simp [mul_add]` or `apply mul_add`

DIFFICULTY: easy

**Lean 4 Proof:**
```lean
import Mathlib.Algebra

theorem distributive_property (a b c : Nat) : a * (b + c) = a * b + a * c :=
  by apply mul_add
```
