1. Since f is polynomial-time computable, there exists a polynomial p such that the time complexity of f is bounded by p(|x|), where |x| denotes the size of the input x.
2. Similarly, for g, there exists a polynomial q such that the time complexity of g is bounded by q(|x|).
3. The composition h(x) = f(g(x)) involves first computing g(x) and then computing f on the result. 
4. The time complexity of computing g(x) is bounded by q(|x|), and the size of the output of g(x) is at most q(|x|) (assuming the output size is polynomially bounded by the input size, which is a reasonable assumption for polynomial-time computable functions).
