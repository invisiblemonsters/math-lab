**Natural Language:**
THEOREM: The composition of a polynomial-time reduction with a polynomial-time computable function is itself a polynomial-time reduction. 2. **Composition**: Define the composed function g(x) = R(f(x)). We need to show that g is a polynomial-time reduction.
3. **Polynomial Time**: The running time of g is the sum of the running times of f and R. Since both are polynomially bounded, their sum is also polynomially bounded (use previous theorems on polynomial sums).
4. **Correctness**: Show that x ‚ąą L‚āĀ ‚á® g(x) ‚ąą L‚āā, which follows directly from the correctness of R and the fact that f is a computable function (preserving the language membership).

LEAN HINT:
