THEOREM: The composition of two polynomial-time computable functions is itself polynomial-time computable, where the polynomial bounding the time complexity of the composition is the product of the polynomials bounding the individual functions.

PROOF SKETCH:
1. **Definition Recall**: A function f is polynomial-time computable if there exists a polynomial p such that the time to compute f(x) is bounded by p(|x|), where |x| denotes the length of input x.
2. **Composition Definition**: Given functions f and g, their composition f ∘ g is defined as (f ∘ g)(x) = f(g(x)).
3. **Time Complexity Analysis**: Suppose f is computable in time p(|x|) and g is computable in time q(|x|). To compute f(g(x)), first compute g(x) in time q(|x|), then compute f on the result in time p(|g(x)|).
4. **Polynomial Bound**: Since |g(x)| ≤ q(|x|) (as the output size is bounded by the computation time), the total time is q(|x|) + p(q(|x|)). This is still a polynomial in |x| because the composition of polynomials is a polynomial.
5. **Conclusion**: Thus, the composition f ∘ g is polynomial-time computable with a bounding polynomial of x ↦ q(x) + p(q(x)), which is a polynomial.

  rintro ⟨f, hf⟩ ⟨g, hg⟩,
  refine ⟨g ∘ f, _⟩,
  split,
  { intro x,
    apply hg.1,
    apply hf.1 },
  apply composition_poly_time hf.2 hg.2
end
