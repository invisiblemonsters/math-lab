**Natural Language:**
THEOREM: For any natural numbers a, b, and c, if a ≤ b, then a + c ≤ b + c.

PROOF SKETCH:
1. **Assume** a ≤ b (i.e., there exists a natural number d such that a + d = b).
2. **Add c to both sides** of the equation a + d = b, resulting in a + c + d = b + c.
3. **Apply the definition of ≤** for natural numbers: a + c ≤ b + c because there exists a natural number d such that (a + c) + d = b + c.

LEAN HINT:
```lean

begin
  -- Unfold the definition of ≤ for natural numbers
  unfold has_le.le at h,
  -- Use the exists.intro to extract the witness d
  cases h with d hd,
  -- Substitute hd into the inequality
  rw hd,
  -- Apply the reflexivity of addition
  apply nat.add_le_add_left,
  -- The rest should be straightforward with simp
  simp,
end
```

**Natural Language:**
THEOREM: For all natural numbers n and m, if n ≤ m, then the length of the list [n, n+1, ..., m] is equal to m - n + 1.

PROOF SKETCH:
1. **Base Case**: If n = m, the list is [n], and its length is 1, which equals m - n + 1 = 1.
2. **Inductive Step**: Assume the statement holds for some n and m where n ≤ m. Consider n and m+1. If n ≤ m+1, then either n = m+1 (trivial) or n ≤ m. In the latter, the list [n, ..., m, m+1] has length (m - n + 1) + 1 = (m+1) - n + 1, which matches the formula.

LEAN HINT:
```lean

import Algebra.Order.Nat

theorem list_length_range {n m : ℕ} (h : n ≤ m) :
  (List.range (m - n + 1)).length = m - n + 1 := by
  induction h with k ih
  case refl => simp [List.range]
  case succ => simp [List.range, ih]
```

**Natural Language:**
THEOREM: For all natural numbers a, b, and c, if a ≤ b, then a * c ≤ b * c.

PROOF SKETCH:
1. **Assume** a ≤ b, which means there exists a natural number d such that a + d = b.
2. **Multiply** both sides of the equation by c: (a + d) * c = b * c.
3. **Distribute** multiplication over addition: a * c + d * c = b * c.
4. **Conclude** a * c ≤ b * c, since d * c is a non-negative natural number.

LEAN HINT:
- Imports: `import Data.Nat.Basic`
- Tactics: `intro`, `apply`, `simp`, `omega`

DIFFICULTY: easy

**Lean 4 Proof:**
```lean
import Mathlib

theorem mul_le_mul (a b c : ℕ) (h : a ≤ b) : a * c ≤ b * c :=
  by induction c
     case zero => simp
     case succ n ih =>
       cases h with d hd
       simp [hd, Nat.mul_succ]
       linarith [ih]
```
