**Natural Language:**
THEOREM: The sum of two polynomial functions over the natural numbers is also a polynomial function. Formally, if f, g : ℕ → ℕ are polynomial functions, then the function h : ℕ → ℕ defined by h(n) = f(n) + g(n) is also a polynomial function.

PROOF SKETCH:
1. **Definition of Polynomial Function**: Recall that a function f : ℕ → ℕ is a polynomial function if there exists a polynomial p(x) with non-negative integer coefficients such that f(n) = p(n) for all n ∈ ℕ.
2. **Sum of Polynomials**: Use the fact that the sum of two polynomials is also a polynomial. Specifically, if f(n) = p(n) and g(n) = q(n), then h(n) = p(n) + q(n), which is a polynomial with coefficients obtained by adding the corresponding coefficients of p and q.
3. **Non-Negativity of Coefficients**: Show that the coefficients of the resulting polynomial are non-negative, as the sum of non-negative coefficients is also non-negative.
4. **Conclusion**: Conclude that h is a polynomial function by definition.

LEAN HINT:
- Imports: `import algebra.polynomial`
- Tactics: `simp`, `ring`, `exact`, `apply`, `intro`
- Consider defining polynomial functions using `polynomial.ℕ[X]` and using the `eval` function to evaluate polynomials at natural numbers.

DIFFICULTY: easy

**Lean 4 Proof:**
```lean
import Mathlib

import algebra.polynomial

def is_polynomial_function (f : ℕ → ℕ) : Prop :=
∃ p : polynomial ℕ, ∀ n : ℕ, f n = polynomial.eval n p

theorem sum_of_polynomial_functions (f g : ℕ → ℕ) :
is_polynomial_function f → is_polynomial_function g → is_polynomial_function (λ n, f n + g n) :=
begin
  intro hf hg,
  cases hf with pf hf,
  cases hg with pg hg,
  use pf + pg,
  intro n,
  simp [polynomial.eval_add],
  rw [hf n, hg n],
  ring,
end

theorem non_negativity_of_coefficients (p q : polynomial ℕ) :
(∀ i, 0 ≤ polynomial.coeff p i) → (∀ i, 0 ≤ polynomial.coeff q i) → (∀ i, 0 ≤ polynomial.coeff (p + q) i) :=
begin
  intro hp hq i,
  simp [polynomial.coeff_add],
  apply add_nonneg,
  exact hp i,
  exact hq i,
end

theorem sum_of_polynomial_functions_is_polynomial :
∀ f g : ℕ → ℕ, is_polynomial_function f → is_polynomial_function g →
∃ p : polynomial ℕ, (∀ i, 0 ≤ polynomial.coeff p i) ∧ ∀ n, f n + g n = polynomial.eval n p :=
begin
  intro f g hf hg,
  cases hf with pf hf,
  cases hg with pg hg,
  use pf + pg,
  split,
  apply non_negativity_of_coefficients,
  intro i,
  cases (classical.em (polynomial.coeff pf i = 0)) with hcoeff hcoeff,
  simp [hcoeff],
  apply add_nonneg,
  apply zero_le,
  apply zero_le,
  intro i,
  apply zero_le,
  intro n,
  simp [polynomial.eval_add],
  rw [hf n, hg n],
  ring,
end
```

**Natural Language:**
THEOREM: The set of all natural numbers less than or equal to a given natural number n is finite.

PROOF SKETCH: 
1. **Definition of Finite Set**: Recall that a set is finite if there exists a natural number m such that the set has exactly m elements.
2. **Bounded Natural Numbers**: For a given n, the set {x ∈ ℕ | x ≤ n} contains all natural numbers starting from 0 up to n.
3. **Cardinality Argument**: Show that this set has n+1 elements by constructing a bijection with the set {0, 1, 2, ..., n}.
4. **Conclusion**: Since n+1 is a natural number, the set is finite by definition.

LEAN HINT: 
- Imports: `import Data.Nat.Basic`
- Tactics: `induction n`, `simp`, `exact`, `use` (for constructing the bijection)

DIFFICULTY: easy

**Lean 4 Proof:**
```lean
import Mathlib
import Data.Nat.Basic

def leSet (n : ℕ) : Set ℕ := {x | x ≤ n}

theorem leSetFinite (n : ℕ) : Set.Finite (leSet n) :=
  by exact Set.finite_Iio (n + 1)
```

**Natural Language:**
THEOREM: For any natural number n, the sum of the first n natural numbers is equal to n(n + 1)/2. Formally, ∀ n ∈ ℕ, ∑_{i=0}^{n-1} i = n(n + 1)/2.

PROOF SKETCH:
1. **Base Case**: Verify the formula for n = 0, where the sum is 0.
2. **Inductive Step**: Assume the formula holds for some arbitrary natural number k. Then, show it holds for k + 1 by adding k to both sides of the inductive hypothesis and simplifying to match the formula for k + 1.

LEAN HINT:
- Imports: `import data.nat.basic`
- Tactics: `induction n`, `simp`, `rw add_comm`, `norm_num`

DIFFICULTY: easy

**Lean 4 Proof:**
```lean
import Mathlib
import data.nat.basic
import algebra.big_operators.basic

theorem sum_of_first_n_natural_numbers (n : ℕ) : ∑ i in finset.range n, i = n * (n - 1) / 2 + n :=
  by induction n with
  | zero => simp
  | succ k ih =>
    simp
    rw [add_comm, ih]
    ring
    norm_num

theorem sum_of_first_n_natural_numbers' (n : ℕ) : ∑ i in finset.range n, i = n * (n + 1) / 2 :=
  by induction n with
  | zero => simp
  | succ k ih =>
    simp
    rw [add_comm, ih]
    have h : (k + 1) * k = k * (k + 1) := by ring
    rw [h]
    ring
    norm_num
    exact rfl
```
