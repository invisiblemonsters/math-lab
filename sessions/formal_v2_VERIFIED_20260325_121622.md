**Natural Language:**
THEOREM: The Cartesian product of two finite subsets of the natural numbers is also a finite subset of the natural numbers.

PROOF SKETCH: 
1. Let A and B be finite subsets of ℕ.
2. By definition of finiteness in ℕ, there exist natural numbers m and n such that A ⊆ {0, 1, ..., m} and B ⊆ {0, 1, ..., n}.
3. The Cartesian product A × B is a subset of {0, 1, ..., m} × {0, 1, ..., n}.
4. The set {0, 1, ..., m} × {0, 1, ..., n} has (m+1)(n+1) elements, which is finite.
5. Since A × B is a subset of a finite set, it is also finite.

LEAN HINT:
```lean

lemma finite_cartesian_product (A B : set ℕ) (hA : finite A) (hB : finite B) : finite (A × B) :=
begin
  -- Use the finite.prod theorem from mathlib
  exact finite.prod hA hB,
end
```

DIFFICULTY: easy

**Natural Language:**
THEOREM: The Cartesian product of a finite set of natural numbers with itself is also a finite set of natural numbers.

PROOF SKETCH:
1. Recall the definition of the Cartesian product: A × A = { (a, b) | a, b ∈ A }.
2. Since A is finite, there exists a bijection between A and the set {1, 2, ..., n} for some n ∈ ℕ.
3. The Cartesian product A × A can be put into a bijection with {1, 2, ..., n} × {1, 2, ..., n}, which has n² elements.
4. Therefore, A × A has n² elements, which is finite.

LEAN HINT:
- Imports: `import data.set.finite`
- Tactics: `apply`, `intro`, `exact`, `simp`, `omega` (for natural number arithmetic)

DIFFICULTY: easy

**Lean 4 Proof:**

import Mathlib
import data.set.finite

theorem cartesian_product_finite (A : Set ℕ) (hA : A.Finite) : 
  (A ×ˢ A).Finite ∧ (A ×ˢ A).card = A.card * A.card :=
begin
  obtain ⟨n, h1⟩ := Set.Finite_iff_exists_nat_bijection.1 hA,
  let f : A ×ˢ A → Fin (n * n) := 
    fun (x, y) => 
      let i := (h1 x).1 
      let j := (h1 y).1 
      Fin.mk (i * n + j) (by omega),
  apply And.intro,
  { apply Set.Finite_iff_exists_nat_bijection.2,
    exists n * n, f,
    intro (x, y),
    simp [f],
    intro (x, y) (x', y'),
    simp [f],
    intro h2,
    cases h1 x with i hi,
    cases h1 y with j hj,
    cases h1 x' with i' hi',
    cases h1 y' with j' hj',
    simp at h2,
    have h3 : i * n + j = i' * n + j' := by omega,
    have h4 : i = i' := by omega,
    have h5 : j = j' := by omega,
    have h6 : x = x' := by simp [h1, h4],
    have h7 : y = y' := by simp [h1, h5],
    simp [h6, h7],
    intro z,
    cases z with i hi,
    let x : A := h1.symm ⟨i / n, by omega⟩,
    let y : A := h1.symm ⟨i % n, by omega⟩,
    exists (x, y),
    simp [f],
    cases h1 x with j hj,
    cases h1 y with k hk,
    have h2 : j = i / n := by omega,
    have h3 : k = i % n := by omega,
    simp [h2, h3] },
  { have h2 : A.card = n := by simp [h1],
    simp [h2],
    have h3 : (A ×ˢ A).card = n * n := by simp [f],
    simp [h3] }
end
