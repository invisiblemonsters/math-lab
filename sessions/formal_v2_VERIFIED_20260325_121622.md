THEOREM: The class of decidable languages is closed under the right quotient operation. Specifically, if L is a decidable language over an alphabet Σ, then the language L / Σ* = { w | ∃ s ∈ Σ*, ws ∈ L } is also decidable.

PROOF SKETCH:

2. **Constructing the Decider for L/Σ***: To decide membership in L/Σ*, on input w, we need to check if there exists a suffix s such that ws ∈ L.

4. **Termination**: Because the decider for L always halts, each simulation of L on ws will terminate. Moreover, since we are looking for the existence of any s, if such an s exists, we will find it in finite time (due to the dovetailing argument). If no such s exists, all simulations will eventually reject, and we can conclude w ∉ L/Σ*.

  (f : L₁ poly_time_reduces_to L₂) (g : L₂ poly_time_reduces_to L₃) : 
  L₁ poly_time_reduces_to L₃ := 
begin
  -- Construct the composition function
  let h := λ x, g (f x),

  -- Prove h is polynomial-time computable
  have h_poly : polynomial_time h := 
    by apply poly_time_compose; exact f.poly_time; exact g.poly_time,

  -- Prove the reduction property
  have h_reduces : ∀ x, x ∈ L₁ ↔ h x ∈ L₃ := 
    by intros x; exact iff.trans (f.reduces x) (g.reduces (f x)),

  -- Conclude the reduction exists
  exact ⟨h, h_poly, h_reduces⟩
end
