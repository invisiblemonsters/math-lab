## Turn 20 — Verified ✓

**Natural Language:**
THEOREM: If a language L is in P, then its complement ¬L is also in P, i.e., P is closed under complementation.

PROOF SKETCH: 
1. **Definition of P**: Recall that a language L is in P if there exists a polynomial-time decidable algorithm (Turing machine) that decides membership in L.
2. **Complement Construction**: To decide membership in ¬L, we can simply run the polynomial-time algorithm for L and invert its output (accept if it rejects, and reject if it accepts).
3. **Polynomial-Time Preservation**: Since the original algorithm runs in polynomial time, inverting its output does not increase the running time beyond a polynomial factor. Thus, ¬L is also decidable in polynomial time.
4. **Closure Conclusion**: Therefore, if L is in P, ¬L is also in P, proving that P is closed under complementation.

LEAN HINT:

  -- Unfold the definition of P

  rcases h with ⟨p, dec_L⟩,
  -- Construct the algorithm for ¬L by inverting the output of dec_L
  use p, -- same polynomial
  -- Show that ¬L is decidable in polynomial time using dec_L
  exact (decidable_of_decidable_complement dec_L),
  -- Apply the theorem that decidability of complement is preserved in P
