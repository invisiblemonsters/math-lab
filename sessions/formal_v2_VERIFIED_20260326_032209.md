**Natural Language:**
THEOREM: For all natural numbers a, b, and c, if a ≤ b, then the product of a and c is less than or equal to the product of b and c.

PROOF SKETCH:
1. **Start with the premise**: Assume a, b, c ∈ ℕ and a ≤ b.
2. **Apply multiplication property**: Use the fact that if a ≤ b, then for any natural number c, a * c ≤ b * c. This follows from the monotonicity of multiplication over natural numbers.
3. **Conclude the inequality**: Directly derive a * c ≤ b * c from the premise and the multiplication property.

LEAN HINT:
- Imports: `import Data.Nat.Basic`
- Tactics: `intro`, `apply`, `exact`, `norm_num`, `linarith`

DIFFICULTY: easy

**Lean 4 Proof:**
```lean
import Mathlib
import Data.Nat.Basic

theorem mul_le_of_le (a b c : ℕ) (h : a ≤ b) : a * c ≤ b * c :=
  by apply Nat.mul_le_mul_left; exact h
```
