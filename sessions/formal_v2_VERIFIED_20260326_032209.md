PROOF SKETCH:

2. **Composition**: Given a polynomial-time computable function f and a polynomial-time decidable language L, consider the language {x | f(x) ∈ L}. We need to show that this language is polynomial-time decidable.
3. **Turing Machine Construction**: Construct a Turing machine M that, on input x, computes f(x) and then checks if f(x) is in L. Since f is polynomial-time computable and L is polynomial-time decidable, the composition of these two processes (computing f(x) and deciding L) can be done in polynomial time.
4. **Polynomial Time Bound**: Let p be the polynomial bounding the time to compute f(x), and q be the polynomial bounding the time to decide L on inputs of length up to p(|x|). The total time for M on input x is p(|x|) + q(p(|x|)), which is still a polynomial in |x|.

- Tactics: `apply`, `use` (for constructing the Turing machine), `simp` (for polynomial arithmetic), `exact` (to reference previous lemmas on polynomial-time computability and decidability).
