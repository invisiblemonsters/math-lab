THEOREM: For all natural numbers a, b, and c, if a ≤ b, then the product of a and c is less than or equal to the product of b and c.

PROOF SKETCH:

3. **Conclude the inequality**: Directly derive a * c ≤ b * c from the premise and the multiplication property.

LEAN HINT:
- Imports: `import Data.Nat.Basic`
- Tactics: `intro`, `apply`, `exact`, `norm_num`, `linarith`

DIFFICULTY: easy

## Turn 6 — Verified ✓

**Natural Language:**
THEOREM: The composition of a polynomial-time computable function with a polynomial-time decidable language results in a polynomial-time decidable language.

PROOF SKETCH:
1. **Definitions**: Recall that a function f is polynomial-time computable if there exists a polynomial p such that for all inputs x, f(x) can be computed in time p(|x|). A language L is polynomial-time decidable if there exists a polynomial-time Turing machine that decides L.
2. **Composition**: Given a polynomial-time computable function f and a polynomial-time decidable language L, consider the language {x | f(x) ∈ L}. We need to show that this language is polynomial-time decidable.
3. **Turing Machine Construction**: Construct a Turing machine M that, on input x, computes f(x) and then checks if f(x) is in L. Since f is polynomial-time computable and L is polynomial-time decidable, the composition of these two processes (computing f(x) and deciding L) can be done in polynomial time.
4. **Polynomial Time Bound**: Let p be the polynomial bounding the time to compute f(x), and q be the polynomial bounding the time to decide L on inputs of length up to p(|x|). The total time for M on input x is p(|x|) + q(p(|x|)), which is still a polynomial in |x|.

LEAN HINT:
- Imports: `import complexity.basic`
- Tactics: `apply`, `use` (for constructing the Turing machine), `simp` (for polynomial arithmetic), `exact` (to reference previous lemmas on polynomial-time computability and decidability).

DIFFICULTY: medium

**Lean 4 Proof:**
```lean
import Mathlib
import complexity.basic

open complexity

theorem composition_polynomial_time_decidable {α β : Type} [Size α] [Size β] 
  (f : α → β) (L : Set β) (hf : poly_time_computable f) (hL : poly_time_decidable L) : 
  poly_time_decidable {x | f x ∈ L} :=
begin
  obtain ⟨p, hp⟩ := hf
  obtain ⟨M, q, hM⟩ := hL
  use (λ x, M (f x))
  use (λ x, p (size x) + q (p (size x)))
  intro x
  simp only [poly_time_decidable, poly_time_computable] at hp hM
  apply hM (f x)
  exact hp x
  apply le_trans (add_le_add (hp x) (hM (f x)))
  simp [poly_time_bounded]
  exact le_refl _
end
```
