THEOREM: For all natural numbers a, b, and c, if a ≤ b, then a * c ≤ b * c.

PROOF SKETCH:
1. **Assume** a ≤ b, which by definition means there exists a natural number d such that a + d = b.
2. **Multiply** both sides of the equation a + d = b by c to get a * c + d * c = b * c.
3. **Apply** the distributive property to show a * c ≤ b * c, using the fact that d * c is non-negative.

LEAN HINT:
- Imports: `import Data.Nat.Basic`
- Tactics: `intro`, `apply`, `simp [mul_add, add_le_add_right]`, `omega`

DIFFICULTY: easy

theorem mul_le_mul_right (a b c : ℕ) : a ≤ b → a * c ≤ b * c :=

    have := mul_add c a d
