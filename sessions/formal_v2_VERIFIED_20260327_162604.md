PROOF SKETCH: 
1. Start with the given inequality a > b.

3. Use the property that multiplication preserves inequality for natural numbers (since c is a natural number, it is non-negative, and thus the direction of the inequality is preserved).

- Tactics: `apply mul_le_mul`, `exact`, `linarith` (to handle the inequality)
