THEOREM: The composition of two polynomial-time computable functions is itself a polynomial-time computable function.

PROOF SKETCH: 
1. **Definitions**: Let `f` and `g` be polynomial-time computable functions, where `f` has a polynomial time bound `p` and `g` has a polynomial time bound `q`.
2. **Composition**: Define the composition function `h(x) = f(g(x))`.
3. **Time Complexity**: Show that the time complexity of `h` is bounded by a polynomial in the input size `|x|`. Specifically, if `g` runs in time `q(|x|)` and `f` runs in time `p(|g(x)|)`, then `h` runs in time `q(|x|) + p(q(|x|))`, which is a polynomial in `|x|`.
4. **Conclusion**: Conclude that `h` is polynomial-time computable by constructing the bounding polynomial explicitly.

LEAN HINT:
- Imports: `import Complexity.Basic`
- Tactics: `simp`, `ring`, `apply`, `intro`, `norm_num`, `aesop`

```lean
import Complexity.Basic

variable (α β γ : Type)
variable [Size α] [Size β] [Size γ]

theorem composition_polynomial_time
  (f : β → γ) (g : α → β)
  (p : ℕ → ℕ) (q : ℕ → ℕ)
  (hp : polynomial_time f p)
  (hq : polynomial_time g q) :
  polynomial_time (f ∘ g) (fun n => q n + p (q n)) :=

  simp [(∘), function.composition],
  have h1 : (f ∘ g) x = f (g x) := rfl,
  have h2 : time (f ∘ g) x = time f (g x) + time g x := by simp [time_composition],
  have h3 : time g x ≤ q (size x) := hq x,
  have h4 : size (g x) ≤ q (size x) := size_bound h3,
  have h5 : time f (g x) ≤ p (size (g x)) := hp (g x),
  have h6 : time f (g x) ≤ p (q (size x)) := by apply le_trans h5 (p_monotone h4),
  have h7 : time (f ∘ g) x ≤ q (size x) + p (q (size x)) := by linarith [h2, h3, h6],
  exact h7
end
```

**Natural Language:**
THEOREM: The composition of two polynomial-time computable functions is itself a polynomial-time computable function.

PROOF SKETCH: 
1. **Definition Recall**: A function f is polynomial-time computable if there exists a polynomial p such that for all inputs x, f(x) can be computed in time p(|x|), where |x| denotes the length of x.
2. **Composition Definition**: Given two polynomial-time computable functions f and g, their composition h(x) = f(g(x)) requires computing g(x) first and then computing f on the result.
3. **Polynomial Time Bound**: Let p_f and p_g be the polynomials bounding the computation times of f and g, respectively. The time to compute h(x) is the sum of the time to compute g(x) and the time to compute f on g(x), which is p_g(|x|) + p_f(|g(x)|).
4. **Polynomial Composition**: Since |g(x)| is bounded by some polynomial in |x| (because g is polynomial-time computable), the composition of the polynomials p_f and p_g yields another polynomial, ensuring that h(x) is computable in polynomial time.

LEAN HINT:
- Imports: `import Complexity.Basic`
- Tactics: `simp`, `apply`, `exact`, `ring` for polynomial manipulations
- Structure: 
```lean

  (hf : poly_time f) (hg : poly_time g) : 
  poly_time (λ x, f (g x)) :=
begin
  -- unfold definitions and apply polynomial composition
  simp [poly_time, hf, hg],
  apply exists.intro (p_f ∘ p_g),
  -- verify the polynomial bound
  intro x,
  exact (hg x).trans (hf (g x)),
end
```

PROOF SKETCH:

2. **Construct a new Turing machine**: For L₁ ∪ L₂, construct a Turing machine M' that, on input x, simulates the Turing machines for L₁ and L₂ in parallel (or sequentially, using the fact that polynomial time is closed under addition and multiplication).
3. **Time complexity analysis**: Show that the time complexity of M' is bounded by a polynomial in the length of x, using the fact that the maximum of two polynomials is still a polynomial.

- Use `exists.intro` to provide the Turing machine for the union

- Apply `is_poly_time.decidable` for L₁ and L₂

```lean
open complexity

  (h₁ : L₁ ∈ P) (h₂ : L₂ ∈ P) : union_lang L₁ L₂ ∈ P := 
begin
  rcases h₁ with ⟨M₁, hM₁, p₁, hp₁, h₁⟩,
  rcases h₂ with ⟨M₂, hM₂, p₂, hp₂, h₂⟩,
  let M' := λx, if M₁ x then tt else M₂ x,
  let p' := λn, p₁ n + p₂ n + 1,

  split,
  { exact λx, by simp [M', hM₁ x, hM₂ x] },
  use p',
  split,
  { exact poly_time.add (poly_time.mul hp₁ hp₂) poly_time.const },
  intro x,
  simp [M', union_lang, h₁ x, h₂ x],
  by_cases h : M₁ x,
  { simp [h] },
  { simp [h] }
end
```
