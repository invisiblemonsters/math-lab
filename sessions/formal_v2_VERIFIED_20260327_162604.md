**Natural Language:**
THEOREM: The composition of two polynomial-time computable functions is itself a polynomial-time computable function.

PROOF SKETCH: 
1. **Definitions**: Let `f` and `g` be polynomial-time computable functions, where `f` has a polynomial time bound `p` and `g` has a polynomial time bound `q`.
2. **Composition**: Define the composition function `h(x) = f(g(x))`.
3. **Time Complexity**: Show that the time complexity of `h` is bounded by a polynomial in the input size `|x|`. Specifically, if `g` runs in time `q(|x|)` and `f` runs in time `p(|g(x)|)`, then `h` runs in time `q(|x|) + p(q(|x|))`, which is a polynomial in `|x|`.
4. **Conclusion**: Conclude that `h` is polynomial-time computable by constructing the bounding polynomial explicitly.

LEAN HINT:
- Imports: `import Complexity.Basic`
- Tactics: `simp`, `ring`, `apply`, `intro`, `norm_num`, `aesop`
- Key Definitions: `polynomial_time`, `function.composition`

DIFFICULTY: medium

**Lean 4 Proof:**
```lean
import Mathlib
import Complexity.Basic

open Complexity

section Composition

variable (α β γ : Type)
variable [Size α] [Size β] [Size γ]

theorem composition_polynomial_time
  (f : β → γ) (g : α → β)
  (p : ℕ → ℕ) (q : ℕ → ℕ)
  (hp : polynomial_time f p)
  (hq : polynomial_time g q) :
  polynomial_time (f ∘ g) (fun n => q n + p (q n)) :=
begin
  intro x,
  simp [(∘), function.composition],
  have h1 : (f ∘ g) x = f (g x) := rfl,
  have h2 : time (f ∘ g) x = time f (g x) + time g x := by simp [time_composition],
  have h3 : time g x ≤ q (size x) := hq x,
  have h4 : size (g x) ≤ q (size x) := size_bound h3,
  have h5 : time f (g x) ≤ p (size (g x)) := hp (g x),
  have h6 : time f (g x) ≤ p (q (size x)) := by apply le_trans h5 (p_monotone h4),
  have h7 : time (f ∘ g) x ≤ q (size x) + p (q (size x)) := by linarith [h2, h3, h6],
  exact h7
end

end Composition
```
