**Natural Language:**
THEOREM: The union of two polynomial-time decidable languages over an alphabet Σ is itself a polynomial-time decidable language. Formally, if L₁ and L₂ are in P, then L₁ ∪ L₂ is also in P.

PROOF SKETCH: 
1. Recall that a language L is in P if there exists a polynomial-time decidable Turing machine M that decides L.
2. Given L₁ and L₂ are in P, we can construct a Turing machine M' that decides L₁ ∪ L₂ as follows:
   a. On input x, M' simulates the Turing machines for L₁ and L₂ in parallel.
   b. If either machine accepts x, then M' accepts x.
   c. If both machines reject x, then M' rejects x.
3. Since the simulation of two polynomial-time machines in parallel still runs in polynomial time, M' is a polynomial-time decidable Turing machine.
4. Therefore, L₁ ∪ L₂ is in P.

- Imports: import complexity.basic
- Tactics: apply, exact, use, have, simp, intro

**Lean 4 Proof:**
```lean
import Mathlib
import complexity.basic

open complexity

theorem union_of_poly_time_decidable
  {Σ : Type} [fintype Σ]
  (L₁ L₂ : language Σ)
  (h₁ : L₁ ∈ P)
  (h₂ : L₂ ∈ P) :
  L₁ ∪ L₂ ∈ P :=
begin
  rcases h₁ with ⟨M₁, hM₁, p₁⟩,
  rcases h₂ with ⟨M₂, hM₂, p₂⟩,
  let M' := machine.parallel M₁ M₂,
  let p' := λ n, max (p₁ n) (p₂ n),
  use M',
  have : M'.eval x = some tt ↔ x ∈ L₁ ∪ L₂,
  { simp [machine.parallel_eval, hM₁, hM₂] },
  use this,
  have : M'.time x ≤ p' (x.length),
  { simp [machine.parallel_time, p₁, p₂] },
  use this,
  apply polynomial_time_bounded,
  apply poly_time_of_big_o,
  exact big_o.max (p₁.is_big_o) (p₂.is_big_o),
end
```
