THEOREM: For all natural numbers a, b, and c, if a ≤ b, then the minimum of (a + c) and (b + c) is equal to a + c. Formally, min(a + c, b + c) = a + c when a ≤ b.

By the definition of the minimum function, if a + c ≤ b + c, then min(a + c, b + c) = a + c. This follows directly from the properties of natural numbers and the minimum function.

THEOREM: For all natural numbers a, b, and c, the minimum of a + c and b + c is equal to the minimum of a and b plus c. Formally, min(a + c, b + c) = min(a, b) + c.

PROOF SKETCH:

2. Case 1: If a ≤ b, then min(a, b) = a. We must show min(a + c, b + c) = a + c. Since a ≤ b, adding c to both sides preserves the inequality (a + c ≤ b + c), so the minimum is a + c.
3. Case 2: If b < a, then min(a, b) = b. Similarly, we show min(a + c, b + c) = b + c, as b + c < a + c.
4. In both cases, the equality holds by the definition of min and basic properties of natural number addition.
