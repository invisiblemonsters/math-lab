**Natural Language:**
THEOREM: For all natural numbers a, b, and c, if a ≤ b, then the minimum of (a + c) and (b + c) is equal to a + c. Formally, min(a + c, b + c) = a + c when a ≤ b.

PROOF SKETCH: We start by assuming a ≤ b. Adding c to both sides of the inequality preserves the order, so a + c ≤ b + c. By the definition of the minimum function, if a + c ≤ b + c, then min(a + c, b + c) = a + c. This follows directly from the properties of natural numbers and the minimum function.
