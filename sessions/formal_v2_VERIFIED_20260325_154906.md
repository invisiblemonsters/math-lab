**Natural Language:**
THEOREM: Polynomial-time reductions preserve membership in NP. Specifically, if a language L₁ is polynomial-time reducible to a language L₂ (denoted L₁ ≤ₚ L₂) and L₂ is in NP, then L₁ is also in NP.

PROOF SKETCH: 
1. **Definitions**: Start with the definitions of polynomial-time reducibility (L₁ ≤ₚ L₂) and NP. 
2. **NP Verification**: For L₂ ∈ NP, there exists a polynomial-time verifier V for L₂. 
3. **Reduction Function**: Use the polynomial-time computable function f that reduces L₁ to L₂. 
4. **Construct Verifier**: Construct a verifier V' for L₁ by composing f with V. Specifically, for input x and certificate c, V' runs f on x to get f(x), then uses V to verify f(x) with certificate c. 
5. **Polynomial Time**: Show that V' runs in polynomial time by using the polynomial bounds of f and V. 
6. **Correctness**: Prove that V' correctly verifies membership in L₁ by the properties of f and V.

LEAN HINT:
```lean
import complexity_classes
import polynomial_time_reductions

theorem np_closed_under_reduction {Σ : Type} [alphabet Σ] {L₁ L₂ : language Σ} 
  (h_red : L₁ ≤ₚ L₂) (h_np : L₂ ∈ NP) : L₁ ∈ NP :=
begin
  -- Unfold definitions and extract verifier for L₂
  cases h_np with V hV,
  -- Extract reduction function f and its polynomial time bound
  cases h_red with f hf,
  -- Construct new verifier V' for L₁
  let V' := λ x c, V (f x) c,
  -- Prove V' is a verifier for L₁ in polynomial time
  apply exists_verifier,
  use V',
  -- Show V' runs in polynomial time using hf and hV
  apply poly_time_comp,
  exact hf,
  exact hV,
  -- Verify correctness of V'
  intros x,
  split,
  { intro hx,
    -- If x ∈ L₁, then f x ∈ L₂, so there exists a certificate
    cases hV (f x) with c hc,
    use c,
    exact hc },
  { intros h,
    -- If V' accepts, then V accepts f x, so x ∈ L₁
    apply h_red.forward,
    apply hV.complete _ h }
end
```
