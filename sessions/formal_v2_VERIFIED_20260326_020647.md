THEOREM: The sum of the first n natural numbers, denoted as 1 + 2 + 3 + ... + n, is equal to n(n+1)/2.

PROOF SKETCH: 
1. **Base Case**: Verify the formula for n = 1, where the sum is 1 and the formula yields 1(1+1)/2 = 1.
2. **Inductive Step**: Assume the formula holds for some arbitrary natural number k. Then, for k + 1, show that the sum of the first k + 1 natural numbers equals (k + 1)(k + 2)/2 by adding (k + 1) to both sides of the inductive hypothesis.

LEAN HINT:
- Imports: `import data.nat.basic`
- Tactics: `induction n with n ih`, `simp [nat.add_comm, nat.mul_comm, nat.add_assoc]`, `omega` (if needed for simple arithmetic verification)

DIFFICULTY: easy

**Lean 4 Proof:**

```lean
import Mathlib
import data.nat.basic

theorem sum_of_first_n_natural_numbers (n : ℕ) : 
  (list.range (n + 1)).sum (λ x, x) = n * (n + 1) / 2 :=
begin
  induction n with n ih,
  { simp, },
  { simp [nat.add_comm, nat.mul_comm, nat.add_assoc] at *,
    rw [list.sum_range_succ, ih],
    ring_nf,
    omega, }
end
```

**Natural Language:**
THEOREM: The sum of the first n natural numbers, denoted as 1 + 2 + 3 + ... + n, is equal to n(n+1)/2.

PROOF SKETCH:
1. **Base Case**: Verify the formula for n = 1, where 1 = 1(1+1)/2.
2. **Inductive Step**: Assume the formula holds for some k (i.e., 1 + 2 + ... + k = k(k+1)/2). Then, for k + 1, show that the sum 1 + 2 + ... + k + (k+1) equals (k+1)(k+2)/2 by algebraic manipulation.
3. **Conclusion**: By mathematical induction, the formula holds for all natural numbers n.

LEAN HINT:
- Imports: `import data.nat.basic`
- Tactics: `induction`, `simp`, `ring`
- Structure: Define the sum function recursively or use existing Mathlib definitions.

DIFFICULTY: easy

**Lean 4 Proof:**
```lean
import Mathlib

def sum_of_first_n (n : Nat) : Nat :=
  Nat.rec 0 (fun n sum => sum + (n + 1)) n

theorem sum_of_first_n_eq (n : Nat) : sum_of_first_n n = n * (n + 1) / 2 :=
  by induction n
     case zero =>
       simp [sum_of_first_n, Nat.rec]
     case succ k ih =>
       simp [sum_of_first_n, Nat.rec]
       ring
       exact ih
```
