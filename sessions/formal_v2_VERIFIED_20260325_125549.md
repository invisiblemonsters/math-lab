**Natural Language:**
THEOREM: The image of a finite set of natural numbers under a polynomial function is also a finite set of natural numbers.

PROOF SKETCH: 
1. Recall that a polynomial function p: ℕ → ℕ of degree d has the form p(n) = a_d n^d + ... + a_1 n + a_0, where a_i are natural numbers.
2. Let A be a finite set of natural numbers. By definition, A has a finite number of elements, say |A| = k.
3. For each element n in A, p(n) is a natural number since the coefficients and n are natural numbers, and the operations of addition and multiplication are closed in ℕ.
4. The image p(A) = { p(n) | n ∈ A } is a set of natural numbers. 
5. Since A is finite and p is a function, p(A) has at most k elements (by the definition of function image), hence p(A) is finite.

LEAN HINT:
- Imports: `import data.nat.basic` and `import data.set.finite`
- Tactics: `intro`, `apply`, `exact`, `simp` using definitions of polynomial functions and finite sets.

DIFFICULTY: easy

**Lean 4 Proof:**
```lean
import Mathlib
import data.nat.basic
import data.set.finite

-- Define polynomial function
def polynomial (d : ℕ) (a : ℕ → ℕ) (n : ℕ) : ℕ :=
  (List.range (d + 1)).foldr (λ i sum, sum + a i * n ^ i) 0

-- Define the property of being a polynomial function
def is_polynomial (p : ℕ → ℕ) : Prop :=
  ∃ (d : ℕ) (a : ℕ → ℕ), ∀ (n : ℕ), p n = polynomial d a n

-- The theorem
theorem finite_image_of_polynomial (p : ℕ → ℕ) (hp : is_polynomial p) (A : Set ℕ) (hA : A.Finite) :
  (p '' A).Finite :=
begin
  -- Extract the degree and coefficients of the polynomial
  obtain ⟨d, a, hpoly⟩ := hp,
  -- Use the definition of finite sets
  apply Set.Finite.image,
  -- Show that p is a function
  exact hpoly,
  -- Show that A is finite
  exact hA,
end
```

**Natural Language:**
THEOREM: The union of two finite subsets of the natural numbers is also a finite subset of the natural numbers.

PROOF SKETCH:
1. Let A and B be finite subsets of ℕ.
2. By definition of finiteness, there exist natural numbers m and n such that A ⊆ {0, 1, ..., m} and B ⊆ {0, 1, ..., n}.
3. Consider the set C = {0, 1, ..., max(m, n)}. Clearly, A ∪ B ⊆ C.
4. Since C is finite, A ∪ B is also finite.

LEAN HINT:
- Import necessary modules: `import data.set.finite`
- Use `set.finite` typeclass and its properties.
- Tactics: `apply`, `exact`, `use` (for existential quantifiers), `norm_num` (if needed for max computation).

PROOF SKETCH:

2. Base case: The union of a single finite set is trivially finite.
3. Inductive step: Assume the union of n finite sets is finite. For n+1 sets, use the fact that the union of two finite sets is finite (previously verified theorem 15) to show the union of the first n sets with the (n+1)th set is finite.

LEAN HINT:

- Tactics: induction, apply, exact, cases, simp [union_finite]

DIFFICULTY: medium

**Lean 4 Proof:**
```lean
import Mathlib
import data.nat.basic
import data.set.finite

open Set

theorem union_finite_of_finite_collection {n : ℕ} (sets : Fin n → Set ℕ) 
  (h : ∀ i, (sets i).Finite) : (⋃ i, sets i).Finite := 
  by induction n
     case zero => 
       simp
       exact finite_empty
     case succ n ih => 
       have : ⋃ i, sets i = ⋃ i, sets (Fin.castSucc i) ∪ sets (Fin.last n),
         by ext x; simp [Fin.exists_fin_castSucc_iff]
       rw this
       apply union_finite
       exact ih (λ i, h (Fin.castSucc i))
       exact h (Fin.last n)
```

**Natural Language:**
THEOREM: The composition of a polynomial function with a linear function results in a polynomial function. Specifically, if p: ℕ → ℕ is a polynomial function and f: ℕ → ℕ is a linear function defined by f(n) = a*n + b for some constants a, b ∈ ℕ, then the function h(n) = p(f(n)) is also a polynomial function.

PROOF SKETCH:
1. Recall the definition of a polynomial function, which is a function of the form p(n) = c_0 + c_1*n + c_2*n^2 + ... + c_k*n^k for some constants c_0, c_1, ..., c_k ∈ ℕ.
2. Substitute f(n) = a*n + b into p(n) to obtain h(n) = p(f(n)) = c_0 + c_1*(a*n + b) + c_2*(a*n + b)^2 + ... + c_k*(a*n + b)^k.
3. Expand the expression using algebraic rules (e.g., binomial expansion) to show that h(n) can be rewritten in the form of a polynomial in n.
4. Conclude that h(n) is a polynomial function because it can be expressed as a finite sum of terms of the form d_i*n^i for some constants d_i ∈ ℕ.

LEAN HINT:
- Import necessary modules: `import algebra.polynomial` and `import data.nat.basic`
- Define polynomial and linear functions using `def` or `structure`
- Use `simp` and `ring` tactics to expand and simplify expressions
- Employ `induction` on the degree of the polynomial if necessary

DIFFICULTY: medium

**Lean 4 Proof:**
```lean
import Mathlib
import algebra.polynomial
import data.nat.basic

-- Define a polynomial function
def isPolynomial (p : ℕ → ℕ) : Prop :=
  ∃ (k : ℕ) (c : ℕ → ℕ),
    (∀ (n : ℕ), c n = 0 → n > k) ∧
    ∀ (n : ℕ), p n = ∑ i in finset.range (k + 1), c i * n ^ i

-- Define a linear function
def isLinear (f : ℕ → ℕ) : Prop :=
  ∃ (a b : ℕ), ∀ (n : ℕ), f n = a * n + b

-- Theorem statement
theorem composition_is_polynomial :
  ∀ (p : ℕ → ℕ) (f : ℕ → ℕ),
    isPolynomial p → isLinear f → isPolynomial (p ∘ f) :=
begin
  intros p f hp hf,
  cases hp with k c hc,
  cases hf with a b hb,
  use (k * (a + 1)),
  use (λ i, ∑ j in finset.range (k + 1), c j * (a ^ j) * (finset.choose j i) * (b ^ (j - i))),
  split,
  { intros n hn,
    simp [hn] },
  { intros n,
    simp [(p ∘ f) n],
    simp [hb n],
    simp [hc.2 n],
    ring,
    simp [finset.sum_range],
    simp [finset.sum_congr],
    apply finset.sum_congr,
    { simp },
    { intros x hx,
      ring } }
end
```
