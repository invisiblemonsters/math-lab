**Natural Language:**
THEOREM: The composition of two polynomial-time computable functions is also polynomial-time computable. Specifically, if f and g are functions from Σ* to Σ* that can be computed in polynomial time, then the function h(x) = f(g(x)) can also be computed in polynomial time.

PROOF SKETCH:
1. **Definition of Polynomial-Time Computable**: Recall that a function f is polynomial-time computable if there exists a Turing machine M that computes f and a polynomial p such that for all inputs x, M halts within p(|x|) steps, where |x| denotes the length of x.
2. **Time Complexity of Composition**: Let p_f and p_g be the polynomials bounding the time complexity of f and g, respectively. We need to show that the time complexity of h(x) = f(g(x)) is bounded by a polynomial in |x|.
3. **Bounding the Composition**: The time to compute g(x) is at most p_g(|x|), and the time to compute f(g(x)) is at most p_f(|g(x)|). Since |g(x)| ≤ p_g(|x|) (as the output length of a polynomial-time computable function is polynomially bounded), we have |g(x)| ≤ p_g(|x|).
4. **Polynomial Bound**: Substitute |g(x)| into p_f to get p_f(p_g(|x|)), which is a polynomial in |x| because the composition of polynomials is a polynomial.
5. **Conclusion**: Therefore, h(x) can be computed in time bounded by the polynomial p_f(p_g(|x|)), proving that the composition of f and g is polynomial-time computable.

- Imports: `import Complexity.Basic`
- Tactics: `simp`, `apply`, `intro`, `norm_num`, `ring` for polynomial manipulations
- Key Definitions: `polynomial_time_computable`, `function.composition`

import Mathlib
import Complexity.Basic

section Composition

variable [Alphabet Σ]

def poly_comp (p q : ℕ → ℕ) : ℕ → ℕ := fun n => p (q n)

theorem poly_comp_is_poly (p q : ℕ → ℕ) (hp : ∃ c n₀, ∀ n ≥ n₀, p n ≤ c * n ^ n₀)
  (hq : ∃ c n₀, ∀ n ≥ n₀, q n ≤ c * n ^ n₀) :
  ∃ c n₀, ∀ n ≥ n₀, poly_comp p q n ≤ c * n ^ n₀ :=
begin
  rcases hp with ⟨cp, np, hp⟩,
  rcases hq with ⟨cq, nq, hq⟩,
  use (cp * cq ^ np),
  use (max np nq),
  intro n,
  intro hn,
  apply le_trans (hp (q n) _),
  apply le_trans (mul_le_mul_of_nonneg_left (hq n _) cp),
  simp,
  ring_nf,
  apply pow_le_pow_of_le_left,
  apply le_trans hn,
  apply le_max_right,
  apply le_max_left,
  apply le_max_left,
end

theorem composition_polynomial_time_computable
  (f g : Σ* → Σ*) (p q : ℕ → ℕ)
  (hf : polynomial_time_computable f p)
  (hg : polynomial_time_computable g q) :
  polynomial_time_computable (function.composition f g) (poly_comp p q) :=
begin
  intro x,
  cases hf (g x) with hf₁ hf₂,
  cases hg x with hg₁ hg₂,
  use (hg₁ + hf₁),
  split,
  apply le_trans (add_le_add hg₂ hf₂),
  simp,
  apply poly_comp_is_poly,
  apply hf,
  apply hg,
  apply le_trans (length_output_le_poly q hg₂),
  apply le_of_eq,
  refl,
end

end Composition
