**Natural Language:**
THEOREM: For any natural number n, the sum of the first n natural numbers is equal to n(n + 1)/2.

PROOF SKETCH:
1. **Base Case**: Verify the formula for n = 0, where the sum is 0, and 0(0 + 1)/2 = 0.
2. **Inductive Step**: Assume the formula holds for some arbitrary natural number k (i.e., 1 + 2 + ... + k = k(k + 1)/2). Then, show it holds for k + 1 by adding (k + 1) to both sides of the inductive hypothesis and simplifying to match the formula for k + 1.

LEAN HINT:
- Imports: `import data.nat.basic`
- Tactics: `induction n with n ih; simp [nat.add_comm, nat.mul_div_cancel]; ring`

DIFFICULTY: easy

**Lean 4 Proof:**
```lean
import Mathlib
import data.nat.basic

theorem sum_of_first_n_natural_numbers (n : ℕ) : 
  (List.range (n + 1)).sum (λ x, x) = n * (n + 1) / 2 :=
by induction n with n ih
   case zero => simp
   case succ n ih => 
     simp [nat.add_comm, nat.mul_div_cancel]
     rw [ih]
     ring

However, a more idiomatic way to write this proof would be:

import Mathlib
import data.nat.basic

theorem sum_of_first_n_natural_numbers (n : ℕ) : 
  ∑ i in range (n + 1), i = n * (n + 1) / 2 :=
begin
  induction n with n ih,
  { simp },
  { simp [nat.add_comm, nat.mul_div_cancel],
    rw [ih],
    ring }
end
```
