**Natural Language:**
THEOREM: The sum of the first n natural numbers, denoted as 1 + 2 + ... + n, is equal to n(n + 1)/2.

PROOF SKETCH:
1. **Base Case**: Verify the formula for n = 1, showing that 1 = 1(1 + 1)/2.
2. **Inductive Step**: Assume the formula holds for some arbitrary natural number k, i.e., 1 + 2 + ... + k = k(k + 1)/2.
3. **Inductive Hypothesis Application**: Demonstrate that if the formula is true for k, it also holds for k + 1 by adding (k + 1) to both sides of the equation and simplifying to show it equals (k + 1)(k + 2)/2.
4. **Conclusion**: Conclude by induction that the formula is valid for all natural numbers n.

LEAN HINT:
- Imports: `import data.nat.basic`
- Tactics: `induction n`, `simp`, `ring`
- Suggested Structure:
```lean
lemma sum_first_n_nats (n : ℕ) : (sum_{i=1}^n i) = n(n + 1)/2
```

DIFFICULTY: easy

**Lean 4 Proof:**

import Mathlib
import data.nat.basic

lemma sum_first_n_nats (n : ℕ) : (nat.antidiagonal n).sum (λ x, x.1) = n * (n + 1) / 2 :=
begin
  induction n with k ih,
  { simp },
  { simp [nat.antidiagonal_succ, nat.sum_antidiagonal_succ],
    have h1 : (nat.antidiagonal k).sum (λ x, x.1) = k * (k + 1) / 2 := ih,
    have h2 : (k + 1) + k * (k + 1) / 2 = (k + 1) * (k + 2) / 2,

end

import Mathlib
import data.nat.basic

def sum_first_n (n : ℕ) : ℕ :=
  (list.range (n + 1)).sum id

lemma sum_first_n_nats (n : ℕ) : sum_first_n n = n * (n + 1) / 2 :=
begin
  induction n with k ih,
  { simp [sum_first_n] },
  { simp [sum_first_n],
    have h1 : sum_first_n k = k * (k + 1) / 2 := ih,
    have h2 : (k + 1) + k * (k + 1) / 2 = (k + 1) * (k + 2) / 2,
    { ring },
    exact h2 }
end

**Natural Language:**
THEOREM: The union of two finite sets is finite, and its cardinality is at most the sum of the cardinalities of the individual sets.

PROOF SKETCH:
1. **Definition of Finite Sets**: Recall that a set is finite if there exists a natural number n such that the set has exactly n elements.
2. **Union Operation**: Consider two finite sets A and B with cardinalities |A| = m and |B| = n.
3. **Cardinality of Union**: The cardinality of the union |A ∪ B| is at most m + n, since every element in A ∪ B is either in A, in B, or in both.
4. **Inclusion-Exclusion Principle**: Use the principle to show |A ∪ B| = |A| + |B| - |A ∩ B|, which is ≤ |A| + |B|.

LEAN HINT:
- Imports: `import data.set.finite`
- Tactics: `simp`, `omega`, `exact`, `apply`, `intro`, `cases`, `linarith`
- Suggested Structure:
```lean
theorem finite_union_cardinality {α : Type} (A B : set α) (hA : finite A) (hB : finite B) :
  finite (A ∪ B) ∧ |A ∪ B| ≤ |A| + |B| :=
begin
  -- Apply finite.union and use cardinality properties
  exact ⟨hA.union hB, by simp [cardinality_union, *]; linarith⟩
end
```

DIFFICULTY: easy

**Natural Language:**
THEOREM: The sum of the first n natural numbers starting from a non-zero value m is equal to (n/2) * (2m + (n - 1)), i.e., m + (m+1) + ... + (m + n - 1) = n * (2m + n - 1) / 2.

PROOF SKETCH:
1. **Base Case**: Verify the formula for n = 1, showing that m = (1/2) * (2m + 1 - 1) simplifies correctly.
2. **Inductive Step**: Assume the formula holds for some k, then prove it for k + 1 by adding the next term (m + k) to both sides and simplifying.
3. **Algebraic Manipulation**: Use basic arithmetic operations and algebraic properties to rearrange and confirm the equality.

LEAN HINT:
- Imports: `import data.nat.basic`
- Tactics: `induction n`, `simp`, `ring`, `exact`, `omega`

DIFFICULTY: medium

**Lean 4 Proof:**
```lean
import Mathlib
import data.nat.basic

theorem sum_of_first_n_natural_numbers (m n : ℕ) : 
  (List.range n).map (λ i => m + i).foldl (λ a b => a + b) 0 = n * (2 * m + n - 1) / 2 :=
begin
  induction n with k hk,
  { simp, exact Nat.zero_mul _ },
  { simp only [List.range_succ, List.map_append, List.foldl_append],
    simp only [List.foldl_nil, List.foldl_cons],
    have : k * (2 * m + k - 1) / 2 + (m + k) = (k + 1) * (2 * m + (k + 1) - 1) / 2,
    { ring },
    exact this ▸ (congr_arg (λ x => x + (m + k)) hk) }
end
However, the above solution uses List.range, List.map and List.foldl which are not very efficient for this problem. Here is a more straightforward solution using mathematical induction:

import Mathlib
import data.nat.basic

theorem sum_of_first_n_natural_numbers (m n : ℕ) : 
  ∃ s : ℕ, (∀ i : ℕ, i < n → s = m + i + (∑ j in Finset.range i, m + j)) :=
begin
  induction n with k hk,
  { use m, intro i, intro hi, exfalso, exact Nat.not_lt_zero i hi },
  { obtain ⟨s, hs⟩ := hk,
    use s + (m + k),
    intro i, cases i,
    { simp },
    { intro hi,
      have : i < k := Nat.lt_of_lt_pred hi,
      simp [hs i this] } }
end

theorem sum_of_first_n_natural_numbers_eq (m n : ℕ) : 
  (∑ i in Finset.range n, m + i) = n * (2 * m + n - 1) / 2 :=
begin
  induction n with k hk,
  { simp },
  { have : (∑ i in Finset.range (k + 1), m + i) = (∑ i in Finset.range k, m + i) + (m + k),
    { simp only [Finset.sum_range_succ] },
    simp only [this, hk],
    ring }
end
```

## Turn 12 — Verified ✓

**Natural Language:**
THEOREM: The sum of the first n even natural numbers, denoted as 2 + 4 + ... + 2n, is equal to n(n + 1).

PROOF SKETCH:
1. **Base Case**: Verify the formula for n = 1, where the sum is 2, and 1(1 + 1) = 2.
2. **Inductive Step**: Assume the formula holds for some k (i.e., 2 + 4 + ... + 2k = k(k + 1)). Then, for k + 1, the sum becomes 2 + 4 + ... + 2k + 2(k + 1). By the inductive hypothesis, this equals k(k + 1) + 2(k + 1) = (k + 1)(k + 2), which matches the formula for n = k + 1.

LEAN HINT:
```lean
import data.nat.basic

theorem sum_first_n_even (n : ℕ) : (2 + 4 + ... + 2n) = n(n + 1) := by
  induction n with
  | zero => simp
  | succ k ih => 
    rw [ih, ← add_assoc, mul_add, add_comm]
    simp [nat.succ_eq_add_one, mul_comm]
```
